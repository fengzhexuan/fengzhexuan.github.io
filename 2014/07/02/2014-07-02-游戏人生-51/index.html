游戏人生

发表于 2014-07-02 | 分类于 随笔

去年看爸爸去哪儿的时候，一开始没啥感觉，只觉得kimi萌萌的，cindy好能跑，石头很懂事，天天么感觉不诚实，王诗龄就傻傻的。第一季的活动都很轻松，做饭为主，找食材为辅。等到第二季感觉几个爸爸过的很辛苦，也许是因为关注了黄磊和他老婆的微博，他们经常在上面po出来的经常是大人或者孩子在睡觉的照片。这年头做人不容易，想上节目出个名博个赞也是体力活，要被摆弄来摆弄去的。

吴镇宇也在微博上po过这么一条：太太帶兒子看完大上海，說兒子知道原來我是壞人很不不開心，我只可跟四歲的他講，爸演壞人是為了賺錢給你買玩具，誰知他說以後不買玩具，我也不能再演壞人！我說好！之後就開始擔心，如果某天演愛情戲，他又投訴我為何抱著的不是媽媽，我怎辦呢？不過他今天已改口說要玩具，爸爸去演壞人吧！太懂事了。

越来越觉得生活的苦，普通人是这样，明星看来也一样。生活越来越像一场游戏，打不完的关卡和小怪兽。
