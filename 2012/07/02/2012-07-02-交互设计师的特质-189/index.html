交互设计师的特质

发表于 2012-07-02 | 分类于 随笔

　　用户体验在国内的互联网环境中基本上渗透率已经较高了，高校也逐渐将交互设计作为一个单独的专业来开设。那么，什么是交互设计师的特质？

好奇心
　　坦白的说，好奇心是一个“设计师”而不仅仅是“交互设计师”必备的特质。生活中的每样东西，不论好坏，几乎都是经过设计的。因此，一个合格的设计师需要充满好奇心：“它为什么是这样设计的，怎样设计会更好？”与此同时，好奇心是设计师的灵感来源。
　　举个简单的例子，国内几乎每个玻璃门上都标有“推”或“拉”的标识，尽管有些门推拉都可以打开。而较为通用的做法则是，装有横着的把手代表可以推，竖着的的把手代表可以拉，如此则较为符合一般人的心智模型。这里要吐槽一下公司的推拉门，每次进出都需要看把手旁边的标识是推还是拉。

同理心
　　要跟谁同理心呢？当然是用户。还记得白鸦和keso在UCD社区上有关“装不装用户”的辩论，我认为同理心是我们的目标，而装不装用户则是达到该目标的一种方法。作为设计师，有没有充分的了解用户的需求，而这才是同理心真正的核心。
　　就我个人而言，平常会接触到很多的产品，而在体验这些新奇产品的同时，却没有从用户的角度去看（有没有真正的去用），而更多的关注设计的亮点，这一点是以后需要弥补与加强的。

结构性思维
　　设计从了解需求到最后定稿，其中必然经过各种各样的设计，怎样将自己的设计理由表达出来是非常重要的。你可能会面临来自产品经理或其他团队成员的质疑，因此训练结构性的思维会让你的设计更为严谨。

　　以上，感谢青云的传授。
