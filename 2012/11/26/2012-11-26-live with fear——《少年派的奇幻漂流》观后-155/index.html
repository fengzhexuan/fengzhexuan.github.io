live with fear——《少年派的奇幻漂流》观后

发表于 2012-11-26 | 分类于 随笔


　　好长时间没有看电影了，下到电脑上的片子也不大想看，正好少年派来了，去大银幕跟前逃避一会儿现实。

　　在进电影院之前，其实故事的大概都可以猜出来，少年和一只老虎在海上漂流。其实这个简单的梗概是无数故事的原型，这只老虎如果是美女，那么就跟泰坦尼克号一样是个爱情故事；这只老虎如果是帅哥，那就跟春光乍泄一样可以是个基情故事；这只老虎如果是个长辈，那么就跟相约星期二一样是个人生哲理故事；这只老虎如果是少年的另一个灵魂，那么就跟搏击俱乐部和黑天鹅一样是个少年精神分裂的故事。在这部片子里，老虎是少年心中的恐惧和家人的记忆，求生怕死的恐惧需要被战胜，噩梦海难的记忆需要被放下，所以漂流的过程就是一个面对现实不断调整自我精神状态的过程。

　　在这个过程中，少年一开始是躲避老虎，到后来把喂老虎作为自己忙碌的目标——也是避免自己被吃掉，这其实就是战胜恐惧的一种最简单的办法。像很多年前知道的一句箴言一样：if you can not run from it, face it.这其中的哲理箴言蕴藏无数，只是我已不愿去细数。因为只有这些箴言内化成你心灵肌肉的一部分，它才能帮助你抵御外在的磨难。

　　其实这也是个讨巧的片子，一开始动物园的各种和海上的各种陆上海上生物，都是制造3D幻觉的最佳道具。这部电影的3D效果和各种CJ及渲染技术的运用比阿凡达要强许多，记得当年电影博物馆巨大IMAx屏幕上奔飞的战鸟那如布片般的翅膀，让人愣愣地直觉得失望。这部片子的视觉效果和音乐都很精致很用心，甚至里面的随喜的佛教徒都是台湾人饰演的，因为中国大陆的环境已然生长不出那样一张清纯无邪的脸庞。想看完结尾的字幕也是想搞明白这个团队到底有多少种工种，有一部分外国人，还有一部分是台湾人。

　　看完这部片子以后很想去看一下小说原著，李安是很会选剧本的导演，断臂山是，色戒也是。理智与情感是先看的小说，再看了李安拍的电影丝毫没有失望的感觉。微博上有不少人觉得还是李安的三部曲最好看，有民国的遗风，少年派显然商业的无法企及当年的水准。是的，可能那三部曲真的与中国人有关，即使卧虎藏龙其实很一般，但一个导演总有拓展自己创作空间的自由，就像陈凯歌拍了赵氏孤儿又拍很现代的搜索一样，只要创作者用了心，这不值得指责。

　　上一次李安的电影也是在大银幕前看的，色戒。这之间穿越了多少时间空间的领悟。也许对李安有心或是因为看过他的访谈，他在纽约学完戏剧以后，六年多都是无事可做，在家当家庭主夫，被夫人养着。如果他的夫人很功利，如果他放弃了自己的梦想，他很难会有今天。也许关注他，仅仅是因为他和中国人有着一样的黄皮肤，有着相似的领悟。

　　总之，这片子不会教人失望。don’t lose hope,it’s a good thing.
