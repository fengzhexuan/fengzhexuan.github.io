最流行CSS3技术Top 10（三）

发表于 2011-05-03 | 分类于 随笔


　　接上次最流行CSS3技术Top 10（二），下面我们介绍Box-shadow与Text-shadow两种技术。

Box-shadow
　　Box-shadow是我们期待已久的特性，该特性使某个元素具有阴影的效果。Box-shadow的参数如下：
box-shadow: position x-offset, y-offset, blur radius;
　　position：的默认值为outset，它的另外一个值为inset。
　　x-offset：正值表示阴影在元素的右面，负值表示在左面。
　　y-offset：正值表示阴影在元素的下面，负值表示在上面。
　　blur radius：该值如果为零，则阴影没有模糊很锐利。
　　 其实，如果大家对Photoshop有一定了解的话，Box-shadow与图层的外阴影的设定几乎一致。
　　现代浏览器全部支持该属性，但4.0以下的Firefox仍然需要添加前缀-moz，在IE9以下的浏览器中可以使用滤镜实现阴影效果。

　　 这样一个完整的Box-shadow声明就变成以下的样子：
.boxshadow {
-moz-box-shadow: 3px 3px 10px #333;
-webkit-box-shadow: 3px 3px 10px #333;
box-shadow: 3px 3px 10px #333;
/ For IE 8 /
-ms-filter: “progid:DXImageTransform.Microsoft.Shadow(Strength=4, Direction=135,Color=’#333333’)”;
/ For IE 5.5 - 7 /
filter: progid:DXImageTransform.Microsoft.Shadow(Strength=4, Direction=135,Color=’#333333’);
}
　　Text-shadow
　　Text-shadow与Box-shadwo是近亲，不过Text-shadow是为文字添加阴影的效果。利用该效果可以使文字生成外发光、浮雕、凸版印刷的外观效果。该特性的参数与Box-shadow基本相同，只不过position没有inset值。现代浏览器全部支持该属性，在IE9以下的浏览器中可以使用滤镜实现该效果。
　　 这样一个完整的Text-shadow声明就变成以下的样子：
.textshadow h2 {
text-shadow:1px 1px 2px rgba(48,80,82,0.8);
-ms-filter: “progid:DXImageTransform.Microsoft.Shadow(Strength=2,Direction=135, Color=’#305052’)”;
/ For IE 5.5 - 7 /
filter: progid:DXImageTransform.Microsoft.Shadow(Strength=2, Direction=135,Color=’#305052’);
}
　　下次将讲解Gradient（渐变）与多重背景这两种属性，敬请关注！
