最流行CSS3技术Top 10（二）

发表于 2011-05-01 | 分类于 随笔


　　接上次最流行CSS3技术Top 10（一），下面我们介绍RGBA与Border-Radius两种技术。

RGBA
　　RGBA是CSS中众多定义颜色方法的一种，唯一不同的是在定义颜色的同时可以设置透明度。RGBA的参数如下：
property: rgba(r color value, g color value, b color value, opacity value);
　　其中color value的值从0至255，opacity value的值从0.0至1.0。现代所有浏览器都支持该属性，包括IE9。但是在IE9以下的浏览器中，需要使用滤镜来实现透明的效果。

　　 这样一个完整的RGBA声明就变成以下的样子：
.rgba {
background-color: #ff0000;
background-color: transparent; / 在IE8中需要使用transparent以使滤镜生效 /
background-color: rgba(255, 0, 0, 0.3);
-ms-filter: “progid:DXImageTransform.Microsoft.gradient(startColorstr=#4CFF0000,endColorstr=#4CFF0000)”;
/ IE8滤镜 /
filter: progid:DXImageTransform.Microsoft.gradient(startColorstr=#4CFF0000,endColorstr=#4CFF0000);
/ IE8以下的滤镜 /
}
　　Border-radius（圆角边框）
　　Border-radius能够实现圆角边框的效果，可以说它是CSS3中最有用的属性之一，让四方的边框变得生动好看起来。当四角的值都相同时可以使用简写。Border-radius的参数如下：
border-radius: top value, right value, bottom value, left value;
　　关于Border-radius在不同浏览器中实现使用不同的前缀，但是在IE9以下的浏览器没有滤镜能够实现相同的效果。最简单的方法是对Border于Border-radius都进行设置，在老旧的浏览器只实现四方的边框。
　　 这样一个完整的Border-radius声明就变成以下的样子：
　　当四角的值相同时
.roundcorners {
-moz-border-radius: 20px;
-webkit-border-radius: 20px;
border-radius: 20px;
}
　　当四角的值不同时
.roundcorners {
border: 5px solid #aaa;
-moz-border-radius: 10px 5px;
-webkit-border-top-left-radius: 10px;
-webkit-border-top-right-radius: 5px;
-webkit-border-bottom-right-radius: 10px;
-webkit-border-bottom-left-radius: 5px;
border-radius: 10px 5px;
}
　　下次将讲解Box-shadow与Text-shadow这两种属性，敬请关注！
