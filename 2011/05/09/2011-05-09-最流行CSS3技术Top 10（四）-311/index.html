最流行CSS3技术Top 10（四）

发表于 2011-05-09 | 分类于 随笔


　　接上次最流行CSS3技术Top 10（三），下面我们介绍Gradient（渐变）与Multi-Background（多重背景）两种技术。

Gradient（渐变）
　　Gradient同样是CSS3中优美特性之一。在CSS3之前，通常实现渐变效果需要背景图片，但在CSS3中让浏览器来渲染渐变的效果，可以节省文件大小，节省带宽。虽然它有如此多的好处，但是需要付出代价，Gradient是一个具有众多参数属性的特性。
　　目前只有Firefox与Chrome支持该特性，虽然IE有实现线性渐变的滤镜，但却不能实现角度渐变。

　　 这样一个完整的Gradient声明就变成以下的样子：
.gradient {
color: #fff;
background: #aaaaaa url(gradient_slice.jpg) 0 0 x-repeat; /在不支持该特性的浏览器中依然使用背景图片/
background-image: -moz-linear-gradient(top, #07407c, #aaaaaa);
background-image: -webkit-gradient(linear,left top,left bottom,color-stop(0, #07407c),color-stop (1, #aaaaaa));
-ms-filter: “progid:DXImageTransform.Microsoft.gradient(startColorStr=’#07407c’,EndColorStr=’#aaaaaa’)”;
filter:progid:DXImageTransform.Microsoft.gradient(startColorStr=’#07407c’, EndColorStr=’#aaaaaa’);
}

　　Multi-Background（多重背景）
　　在CSS3中，你可以同时使用多个背景。该特性的参数非常简单，只需要用逗号将不同的图片分隔开来。但需要注意的一点是，只有在最后的图片中可以设定背景颜色。它的参数如下：
background:
image position size repeat attachment box,
image position size repeat attachment box,
image position size repeat attachment box,
image position size repeat attachment box color;

　　最新的浏览器都支持该特性，IE9还同时支持多重背景颜色。如果想要在所有浏览器中实现相同的效果，你需要在不支持该特性的浏览器中使用合成后的背景图片。
　　 这样一个完整的Multi-Background声明就变成以下的样子：

#wrap {
background: url(body-full.gif) top left no-repeat;
background:
url(body-top.gif) top left no-repeat,
url(body-bottom.gif) bottom left no-repeat,
url(body-middle.gif) left repeat-y;
-ms-filter: “progid:DXImageTransform.Microsoft.AlphaImageLoader(src=’ body-top.gif’,sizingMethod=’scale’)”;
filter: progid:DXImageTransform.Microsoft.AlphaImageLoader(src=’body-top.gif’, sizingMethod=’scale’);
zoom: 1;
}

　　当然你也可以这样写：
body {
background: url(all_layers.jpg) no-repeat 0 0;
background-image:url(first_layer.png),url(second_layer.png),url(third_layer.png),url(fourth_layer.png);
background-repeat: no-repeat, no-repeat,no-repeat, no-repeat;
background-position: 0 0, 0 50, 0 100, 0 200;
-ms-filter: “progid:DXImageTransform.Microsoft.AlphaImageLoader(src=’ body-top.gif’,sizingMethod=’scale’)”;
filter: progid:DXImageTransform.Microsoft.AlphaImageLoader(src=’body-top.gif’, sizingMethod=’scale’);
zoom: 1;
}

　　关于多重背景更好玩的应用可以去阅读 有趣的CSS3多重背景。
