Web 3.0设计简介

发表于 2011-06-17 | 分类于 随笔


　　这是一个Web 2.0的时代，是信息爆炸、快速传播与分享的时代，是社交媒体的时代。看国外，从Youtube到Facebook，再到Twitter；看国内，从博客到人人网，再到微博。还记得Web 1.0的年代么？将信息放到网站上展示，通过邮件进行私人交流。互联网发展将近20年，Web 3.0已经呼之欲出，那么Web 3.0的世界将会什么样的呢？

Web 3.0是什么？
　　很难说清互联网如何发展出来像Facebook和Twitter这样的顶级网站。当你研究Web 3.0的时候，你会发现用户与网络的交互是注定要发生的事情。在Web 2.0中，我们将注意力集中在用户之间的交互或者交流上，现在我们将更加关注用户本身，这是永远不会错的。
　　专家将Web 3.0定义为语义网（Semantic Web），语义的意思是数据驱动。网络收集用户的数据，使其从根本上满足用户的需要。如果你经常搜索“设计博客”，你将会收到很多与设计关联的广告。又或者，当你搜索“电脑”，网络知道“哦，你经常搜索设计”，这样就会给出将“设计”与“电脑”结合的搜索结果。
　　Web 3.0的益处
　　Web 3.0最NB之处是你可以在任何地方获取数据，这得益于各种智能移动终端以及其背后的云计算。想想各大公司都对云计算这块蛋糕在虎视眈眈着，IBM、Amazon、Apple、Microsoft等。物联网也将对Web 3.0做出一些支撑，自我感觉虽然有交集但是服务对象仍然有差别。
　　Web 3.0对设计的影响
　　对于设计师来说，Web 3.0将会有什么影响呢？Web 2.0时代的设计建立在吸引、说服用户的基础上，如此，为了让用户能够在线小费做出了很多努力。你想买这个吗？那赶快点击这个按钮吧。这就是Web 2.0的设计准则。
　　众多的设计元素被添加到Web 2.0的设计中来，比如线性渐变是不可或缺的元素之一。不管你想要怎样的颜色组合，从背景到按钮都存在线性渐变。其他的设计趋势，比如各种徽章、圆角、图标。那么在3.0时代设计师该怎样做呢？
　　Web 2.0 设计 vs. Web 3.0 设计
　　如果说Web 2.0设计是为网络设计的话，那么Web 3.0不仅要考虑网络与浏览器，而要考虑到所有的媒体。随着智能手机与平板电脑的迅速普及，用户想要通过这些终端获取更多的数据、得到更多的服务与应用。现在很多网站为了满足不同的需求，同时维护桌面与移动终端的网站，如何让同一网站能够适应不同的环境则是Web 3.0的一大趋势，比如 响应式设计。
　　同时，让事情变得简单仍然将是设计师专注的目标之一。设计师完全有能力引导用户浏览页面的先后次序。不论何时，设计师要让用户关注其喜欢的内容，而不应分散其注意力。Logo与导航栏也同样是不可忽视的设计要点。
