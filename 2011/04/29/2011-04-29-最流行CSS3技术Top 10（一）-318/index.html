最流行CSS3技术Top 10（一）

发表于 2011-04-29 | 分类于 随笔


　　时下HTML5与CSS3持续火热，不仅最新的浏览器对其都有了不同程度的支持。这些最新技术的应用不仅可以实现各种特殊效果，而且可以节省带宽。这里列出了CSS3中最流行的10种技术。

1. @font-face（服务器端字体）
　　@font-face允许用户从服务器上下载站点所用字体到个人计算机上，从而正确对页面进行渲染。其格式如下：
@rule-name {
font-family: “Fontname”;
src: url(fontfilename.ext) format(‘formatname’);
}

　　@font-face是唯一一个目前所有浏览器都支持的CSS3特性，不过IE9一下的浏览器只支持.eof格式的字体，IE9增加了对.woff格式的支持。不过有两个bug需要注意。在IE9以下的浏览器中，即使正确的声明了@font-face属性，还是不能够正确的渲染字体，要解决这个问题，需要对@font-face进行两次声明，第一次是.eof自己，然后按照@font-face的标准格式声明。在WebKit内核浏览器中同样有个bug，当对@font-face中声明的字体应用font-weight: bold 或者 font-style: italic 时不会起作用，解决方法是将font-weigh，font-style:normal，font-variant属性同时设定为normal以确定一个基准。
　　 这样一个完整的@font-face声明就变成以下的样子：
@font-face {
font-family: ‘Colaborate Light’;
src: url(‘ColabLig.eot’);
src: local(‘☺’), url(‘ColabLig.woff’) format(‘woff’),
url(‘ColabLig.ttf’) format(‘truetype’);
font-weight:normal;
font-style:normal;
font-variant:normal;
}
　　2. Opacity（透明度）
　　顾名思义，你可以对一个元素的透明度进行控制，从完全透明到不透明。透明度的属性会被子元素继承，如果想要达到更加精细的控制效果，则需要使用RGBA属性，在下一小节中我们会介绍到。
opacity: ;
　　其中，value的范围从0.0到1.0。但令人失望的一点是IE浏览器完全不支持Opacity属性，但微软特别提供了一个滤镜效果来实现透明度的改变。较短的写法如下所示：
filter: alpha(opacity=40);
下面是扩展的写法：
filter: progid:DXImageTransform.Microsoft.
Alpha(opacity=40);

使用IE的前缀写法如下：
-ms-filter: “progid:DXImageTransform.
Microsoft.Alpha(opacity=40)”;

　　 这样一个完整的Opacity声明就变成以下的样子：
.opacity {
background-color: #3C4C55;
color: #fff;
opacity: 0.5;
-ms-filter: “progid:DXImageTransform.Microsoft.
Alpha(opacity=20)”;
/ 上面一行只在IE8中有效 /
filter: progid:DXImageTransform.Microsoft.
Alpha(opacity=20);
/ 上面这一行对IE9以下均有效 /
filter: alpha(opacity=20);
}

　　下次将讲解RGBA与Border-radius这两种属性，敬请关注！
