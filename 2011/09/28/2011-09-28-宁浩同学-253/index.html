宁浩同学

发表于 2011-09-28 | 分类于 随笔

在五点半之前到了门口，人还是挺多

宁浩穿着面袋裤来到了定福庄来看望俺们

虽然整个讲座没那么高潮迭起

但也不算沉闷

怎么说呢

感觉宁浩就是一个学以致用的导演

看了盖里奇的电影

学了点东西

拍了了个小成本电影获得了成功

于是拿挣来的钱继续拍了一个《疯狂的石头》之类型片《疯狂地赛车》

原以为他会是很离经叛道的一个人

但是他压根是一个山西人

土土的实实在在的在走自己的路

如此而已

只不过保持了正常人应该有的判断力和思考问题的能力

仅此而已

虽然他并不是有魅力的大叔

但仍是我尊敬的导演

期待他的《黄金大劫案》

希望跟这个时代有关

# 宁浩
赵忠祥老师
经过-平遥
