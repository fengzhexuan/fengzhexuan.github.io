当您见到了您的偶像

发表于 2011-03-31 | 分类于 随笔

冲着吴虹飞去看南周的访谈，结果她被堵在路上一直没有来，所以我不停滴朝门口张望。

堵在朝阳路上快一个钟头的时候，她才来。

终于见到了吴虹飞，她唱过一首歌——《仓央嘉措情歌》，这个名字出现在那里比非2要早得多。

她穿着一件像婚纱的裙子，黑色小西装，从进来以后一直在低头拿着手机发短信还是做什么。

说起话来声音很动听，我以为是平常女人的粗粝嗓音，没想到她的声音本身就像玻璃一样是易碎的玩具。比我想象的要娇小，要年轻。

由于是南周的专场，八个记者都在滔滔不绝采访人物的经历，所以当台下有人问起吴虹飞关于歌手身份的时候，她回避了。

从她的歌声里能读出的只有对出嫁对婚礼近乎绝望的盼望。

观众里冲着她来的人很多，欣赏她的聪明和才华。

她说的话很少，关于北京万恶的交通，她说：“商女不知亡国恨。。”

当您见到了您的偶像。一生中有多少次这样的机会？那样的相遇又会有什么样的尾巴留在你心中。

导航栏设计准则

发表于 2011-03-29 | 分类于 随笔

　　本文翻译自Curing the Navigation-itis Syndrome。

　　网络中每天都有海量的内容被发布，随之而来的问题是如何对这些内容分类，而内容和分类最终将导致导航问题。有时，当导航有太多选项时将会是个令人头疼的问题。

　　所以，治疗导航栏综合症（cure the navigation-itis syndrome）是非常重要的，使得导航栏对用户友好并容易操作。

　　本文提供了一些设计导航栏的技巧。帮助你解决导航问题并体统清晰的用户体验，并用真实的例子教你如何治愈导航炎综合症。

　　但首先，是这些技巧。

设计导航栏的技巧

　　当我们撰写这篇文章之前，Jon (SpyreStudios founder) 和我交换了一些想法，总结出了关于导航栏应该如何放置、如何作用和表现。

1.坚持一个导航菜单

　　通常一个导航菜单就足够了。不要增加不必要的导航条，而用下拉菜单代替。

2.清晰、简单、明显的菜单选项

　　使用清晰、简单易懂的文本，尽可能简单并能表达清楚。这样用户不会感到困惑，且知道他们点击的是什么。

3.保持导航看起来是菜单

　　一个导航菜单应该看起来像一个导航菜单。没有必要重新发明轮子。如果没坏，就不要修。基本上，你的创意应该来自内容，而不是如何呈现内容。相比如何到达内容人们更关心内容本身。所以让人们通过一个不用大惊小怪、可立即识别的过程来得到它。

4.不要使用多于两级的下拉菜单

　　不要使用多于两级的下拉菜单，除非特别必要。如果你需要使用多于三层的下拉菜单，那你可能需要重新返回绘图板。减少菜单项目——大胆裁员。你是否真的需要那两个分类或者页面，也许合二为一更好？这都是关于如何对你的内容做出更好的分类。

5.下拉菜单中不要多于10-12个选项

　　切勿在下拉菜单中放置多于10-12个选项。如果这样打不到你的要求，请重回绘图板，像第4条中那样。

6.最小化下拉菜单

　　将下拉选项保持在最小状态。如果可能，避免使用大型的下拉菜单。更多的选择往往意味着更少的动作。别让用户不知道如何如何在菜单中选择，结果导致用户离开网站。

7.不要只往菜单上放一个图标

　　图标是很重要的，但是如果菜单上只有图标的话，不是每个人都了解图标的含义。三十岁的人会知道闪存的图标代表“保存”。而小孩儿可能会不明白这个含义。

8.不要设计垂直导航栏

　　虽然设计一个垂直的导航栏没有那么难，但是在网页上使用水平导航栏是更加容易并且司空见惯的。垂直导航栏会很不方便，会让一部分用户望而却步，而且会浪费宝贵的屏幕空间。水平导航栏会为网页锦上添花，而垂直导航栏会给你的用户带来麻烦。另外，垂直导航栏会需要更多的空间来让自身变得醒目。这也因项目而异。

9.让你的设计方便触摸屏用户使用

　　触摸屏技术已经被广泛采用，所以你要让自己的导航栏方便触摸屏电子产品（比如IPAD）用户的使用。尤其对于下拉菜单而言，让它们更加容易被点击，而不是只能使用光标停留。

治愈“导航栏迷恋症”：一个实例

　　看看美国国家地理的网站，数一数这个网页有几个导航栏呢？

　　有三个，是的！对于访客是不是有点多呢？如何避免在一个网页上使用三个导航条呢？事实上，把三个导航条合并成一个导航条也是可能的！

1.设置下拉菜单

　　最最重要的是，这个三级菜单的次级菜单（第二级菜单和第三级菜单）可以放到下拉菜单中。这样既可以节省空间，又可以让用户的选择更加明了。比如说如果你对摄影感兴趣的话，你将鼠标悬浮便可以看到关于摄影的所有内容。

2 把搜索框放入现有的导航栏

　　把搜索框单独拉出来是不合理的做法。如果把搜索框放进导航栏，可以节省更多的空间。这样做，呈现给用户单一又实用的导航栏。

3 减少给用户的选择

　　从菜单中砍掉一部分内容会让你的导航栏既方便又实用。如果点击logo可以转到主页的时候还需要再单独做一个主页的链接吗？做次级菜单的时候也要考虑到这一点。

解决导航问题，提供清晰的用户体验

　　如果用户不能访问到，有再好的内容也只是一文不值。好吧，他们最终将找到并访问它。但是很多用户会变得沮丧或者放弃并离开网站，如果他们不能轻易的找到或者因不必要的选择而不堪重负。所以，为什么不更好的提供便利。

　　让用户方便使用和分享内容是治疗导航栏综合症的最好方法。解决了网站的导航问题能给用户提供更清晰的用户体验，让更多用户阅读使用并分享网站的内容而使你快乐。

　　到你了：你学会如何治愈导航栏综合症了么？如果你有更好的技巧和准则，欢迎在下面的评论区域中分享。

逝去的南锣鼓巷

发表于 2011-03-27 | 分类于 随笔

　　地铁八号线终于在南锣鼓巷的南头撅了个口子。

　　第一次听说南锣鼓巷还是在2007年，那时老是骑着自行车在紫禁城周边的大小胡同转悠，想感受最原生态的北京市井文化。那时，前门大街已经开始了改造，成了现在的这幅模样，这里不评判这样的商业模式是否是一个两全其美的方法。

　　南锣鼓巷与前门大街的改造是不一样的，这里没有人们耳熟能详的商业品牌，而是各种充满文艺气息的个性小店——饰品、服饰、小吃、、酒吧、咖啡厅。当然这里是文艺青年经常光顾的地方，还有像我这样的好奇分子。

　　第一次去这条巷子大概是去年的这个时候，和右撇子一起。她说，你去了那么多胡同，没去过南锣鼓巷么？我说，那就去看看吧。我们从巷子的北头进去，两面排布着各式的特色小店。走了不久就看见文宇奶酪店前排着长长的队伍，看见店面告示牌上写着的营业时间——11:00至售罄。其实从巷子的北头走到南头用不了多久，如果你流连于各式小店中，那时间就是个未知数了。

　　到现在，我们共去过四次南锣鼓巷。上次去的时候，南头已经被拆掉了。今天又去了次，来巷子玩的人已然没有去年现在的多了。也许巷子里的各种创意文化已经不再流行，文艺青年也看多了这样的东西，就像798一样。

　　面对南锣鼓巷总是有种复杂的心情，还有其中所蕴含的文化，理不清，说不出口。

费了好几天的时间终于把鲍勃迪伦的传记电影——no direction home给看完了，苹果出品的东西果然追求极致脱俗。

本以为这个电影会是一个人的流水账日记，从年轻到年老，从木吉他到电子吉他，从民谣到流行。但其实这部电影是鲍勃迪伦作为一位音乐冒险家的某一时期的面相——这个面相足以说明鲍勃迪伦的音乐态度——不断寻找，从不停歇。

在鲍勃迪伦为吉他插上电以后，大家叱责他是一个叛徒，其实他只是一个跨界歌手。没有人需要被刻上某个记号，并终身划地为牢。他勇敢地大步向前，用传统的吉他和口琴做音调，顺畅地表达着自己的思考。

有人说：人生的每个阶段都可以用一首迪伦的歌作为注脚。

like a rolling stone隐隐表达着那来自异国的乡愁。每个人都已出发，却还未抵达，举目回望，却找不到属于自己真正的家。只能继续路迢迢，走四方。

RockMelt社交浏览器初探——为Facebook而生

发表于 2011-03-23 | 分类于 随笔

RockMelt是一款专为Facebook设计的社交浏览器，基于chromium开源内核。在此为大家简单介绍一下，当然如果你想使用其社交功能的话，在国内必须得翻墙了，悲催的GFW。

如果不登录脸谱的话该浏览器与一般的浏览器是一样的。

其登录Facebook界面

在登录与未登录状态之间切换都会把之前打开的页面关闭，感觉这一点不很合理。

成功登录之后，左面显示Facebook中的好友，并显示其现在的状态。右面一侧是APP栏，可以添加诸如Youtube，twitter以及Feed各种应用，方便快捷。

另外不得不提的是社交中最重要的功能——分享。地址栏右侧的share按钮可以很方便的将网页分享至Facebook或者是twitter。

有一点疑问，如果在1024*768分辨率的显示器上使用该浏览器，两边的侧边栏会占用不小的空间。
