一瞥

发表于 2014-12-17 | 分类于 随笔

前一段冬天刚来的时候，有一天吃饱了忽然想起来微博上看到过的免费开放的史家胡同博物馆，怀着久违的没出门的热情和兴奋到了史家胡同附近。

史家胡同头上就是北京最好的小学——史家胡同小学。沿着胡同往里面走首先是邓颖超题名的一个X园，紧挨着的院子是有名的史家胡同51号院——洪晃她妈妈的院子。再往里面走是一些单元楼，再走一截就到了胡同博物馆。

参观完了有些失望。这是民国女作家凌淑华家的院子，因为他们夫妇远走英国，儿女们撺掇英国某基金会弄出来的景点。院子里的房子已经变的不中不洋，展览里还扯进来一些根本与此院落无太大关系的徐悲鸿等人，而且附上的展览文字一看就是从百度百科上扒下来的，让人顿觉展览的粗糙。

四合院肯定是很好的，博物馆里的院子有一棵老法国梧桐，他可能才是这小院子沧桑变化的见证者吧。

往回走的时候又路过51号院，忍不住凑到门缝跟前往里面瞧。也许对四合院胡同还有认知和欣赏的我们是最后一代了，虽然没有亲身在胡同里生活过，但是看到胡同和旧院落是由衷的欣喜，因为我们知道那才是真正的北京的内核，即使这颗内核会被岁月剥蚀被政治经济的变化掠夺。

再过些日子的时候，下载了洪晃的《廉价哲学》，忽然想起了洪晃自传性质的一本书——《我的非正常生活》，第一次看这本书是好几年前的事儿了，第一次看洪晃的书则是在上大学的时候了，那时候刚刚知道学校的图书馆的四五层有一些新书，其中就包括洪晃的《无目的美好生活》。这几本书大致的内容都是一样，写自己的从业经历，写自己身边的人，写社会的一些新鲜事，还有一些情感问答Q&A。扯回到《廉价哲学》，看了几页心情轻松了起来。洪晃是见过世面但是又不装的人。然而适合她发光发热走红的土壤已经不在，就连史家胡同的房子也被她继父的前外交部高官儿子抢走了。

存在过，也许就是美好的。就像这下午的太阳，都要过一会儿换个地方探探身子才能够得到。

两部电影

发表于 2014-12-14 | 分类于 随笔

2007年的时候看过一部电影，是《这个男人来自地球》，几乎全片的室内人物对话让观者从无聊慢慢变得越来越不能从影片中回到现实。这部小成本电影颠覆了我对科幻片的认知，此类型的影片往往以宏大的场景、绚丽的特效刺激受众，而《这个男人来自地球》却用相反的手法把科幻的主旨娓娓道来。

几天前，又看了一部拍摄手法非常相似的电影，《彗星来的那一夜》。同样是几乎全片的室内场景，大量的人物对话，讲述了由相干性这一物理概念引申出的科幻故事。虽然剧情不同，但给我的感受却极其相似，从片子开头的无聊到越来越精彩无法自拔的情节，当片尾曲响起的时候还依然沉浸在影片里。

这两部电影的中文译名的质量都差强人意，只看影名的话一定会让你错过它们。

为网站挑选完美图片

发表于 2014-12-10 | 分类于 随笔

在这个看脸的世界，网站的设计越来越漂亮精致，而图片又会占据网页中很大的版面，所以如何挑选合适的图片对于设计师来说也非常重要。最近看到这样一篇文章，列出了10条在选择图片时需要注意的细节，编译在此。

#1 情绪

饱含情感的图片能够影响用户的品牌感知。一张悲伤主题的图片，可能会让用户沉浸在悲伤里。所以问题来了，触达他们的情感是你想要的吗？所以在找寻合适的配图前要确定需要表达怎样的情感。

仔细观看上图，你会感受到一系列的情绪。然而像图片中人物所表现的快乐满足，一定是你最先感受到的。这就是图片传达情感的原理，也是设计网站时为什么需要类似情感图片的原因。

#2 震撼
众所周知，震撼的图片能吸引眼球。不过只有震撼是不够的，完美的图片还要具有强烈的相关性，不然虽然能博得眼球但不能起到传达目的的作用。其目的是为了确保品牌的信息通过图像送达用户。

Poco People选择的图片非常棒，上图是一张放置在荒野的办公桌。

#3 魅力/魔力
充满魅力的图像有一种罕见的粘性。访问过 Surfers Again Sewage一段时间后，我们仍记得这个独特网站的图片。这样的网站配图让人们想得更多，让余音绕梁N日。

上图让你联想到保护海洋和沙滩，远离污水、船舶垃圾等。

#4 正向激励
配图不仅要好看，还要说服使用者对网站提供的产品或服务做出正向的决定。所以必须让人们认为图像的焦点是网站的组成部分。

像 Nike一类的运动服饰网站，采用了穿着特定产品的模特图片。访问网站的用户更可能购买这件特定的产品。

#5 永恒
除了特定的推广图片或者Banner，其他的网站用图不会经常更换。除非打算重新设计网站。所以网站图片需要经得起时间的考验。

Bear CSS的网站图像，能够在一段时间内都觉得不错的原因是，图片里的吉祥物有着非常显著的形象。吉祥物在某种角度来说是常青的，因为在设计吉祥物的时候已经在这方面经过慎重考虑。

#6 瞄准品牌
完全符合品牌、产品/服务、原则、价值、目标的图像是最佳的，它不会留下多余的想象空间。

#7 不让用户分神
有时候打开一个网站并且被其上的图片带入，你会纳闷这是一个关于什么的网站。这就意味着图像已经把你从网站所要传达的焦点中带走了。

#8 代表目标受众的人物形象
设计的网站是为了目标用户，这个是街边卖煎饼的大姐可能都知道的原理。所以，为什么不用一些代表你目标受众的人物图片呢？这些图像将与访问者产生深深的共鸣，而且能提高网站的信任与信誉。

#9 图像一致性
网站设计是一种用图像讲述故事的形式，逐步说服用户去尝试特定的产品或服务。因此图像是关键的设计元素，有助于推动这个故事向前发展。这就是为什么保持网站图像的一致性能让网站以及品牌在目标受众的心中留下深刻印象。

#10 质量
最后，这一切都归结到图像的质量。每一个像素都是完美的，大尺寸，满足最高的清晰度。你可能会觉得这是自言不明的，但往往有些设计师会在这上面出错。这点再如何强调都不为过。

编译自 Noupe。

北方的冬天

发表于 2014-12-04 | 分类于 随笔

这几天一直间歇地刮风，所以没有雾霾。

有暖气的冬天特别怡人。进门扑面而来的温暖让人心中一喜进而感恩，这种感恩也许是因为经历过了湿寒的南方的冬天才有的吧。

还是适应北方的气候，甚至喜欢这种挟风而至的凛冽。呼呼的在耳边刮过心头还是镇定的。那种湿淋淋的下雨冬天心情薄的近乎没有。这种干燥的大面积的呼啸掠过头顶，心里还是果决的欢喜。

对于这里的一切还是熟悉，秋天从旋转门透过玻璃撒到地上的一抹黄色的夕阳是熟悉的，冬天路上那堆落叶碾成的细末是熟悉的，走在路上会想起从前，坐在屋子里会想起回龙观某个天桥台阶上脏兮兮硬邦邦的泡沫颗粒，坐在公交车座位上从屁股上开始蔓延的凉意，总是会提示起无数个在公交车里瑟缩的清晨。关于冬天的回忆总在某一个独处的时刻大面积降临。你好像穿过这些回忆，又好像没有跨过它们，你好像已经走了很远，又好像从来没有离开过。

回到北方居然更加想家，想念那个小镇子的街道，想念那个破落中学的上课下课铃声。想家才让我反省出来我的不自由。虽然回了家也是无事可做更加不自由。

在冬天我们可以做些什么让自己让别人感到温暖，也许只有陪伴。

Android 5.0

发表于 2014-11-29 | 分类于 随笔

最近完整地把Material Design（需翻墙）的spec阅读一遍，把读后的心得记录在此。

设计语言

　　Material Design是Google第一次把一套完整的设计语言形成Spec。不过难以预计的是这次的设计语言会在多大程度上被互联网公司、个人开发者所接受与使用，尤其在众多定制化ROM百花乱放的现状下。
　　简单来说Material Design即让1dp厚度的控件如何通过各自的阴影来表现层级关系。Spec非常详细地描述了大至设计由来小至界面标注动画曲线等等，甚至用大量的Dos&Don’ts来一一举例说明。
　　此外，Material Design是一套跨平台的设计解决方案，不仅针对手机、平板，桌面设备也被纳入。Inbox这一明星产品已经进行了尝试，预计不远的将来更多的产品及设备都会更新。

Floating action button（悬浮按钮）

　　spec建议Floating action button作为Primary action button（首要操作），可以浮动于界面的底部或置于标题与列表之间作为区分。它的功能以及交互逻辑与Path的圆形加号按钮没有本质区别，比较让人费解的是在inbox应用中点击该按钮展开更多功能后却不是再次点击相同位置关闭菜单，而是放置了compose的操作，习惯了Path操作的话会直接导致误触。另外spec里给出的文件列表示例中，通常情况下，“浏览文件”与“新建文件”比较难确定哪个是Primary action，所以采用该示例会有些牵强。

Animation（动画）

　　5.0spec中用一整章的篇幅来说明动画，足见官方对这一版本中动画的重视程度。如果没记错的话，4.x及之前Android OS版本自带的动画效果没有非常明显的动画曲线，可以说基本是线性动画。spec中分别通过Authentic motion（真是动画）、Responsive interaction（响应交互）、Meaningful transitions（有意义的转换）、Delightful details（令人愉悦的细节）来说明动画应遵循的原则。
　　令人印象深刻的是，有意义的转换中的示例是动画中的亮点。尤其是Google calendar中添加提醒事件的转场效果，充分的用有效的动画效果完成界面的跳转。

Style（风格）

　　经过如此多版本的变化，Android OS的风格越来越趋于明亮、简洁。5.0版本中一个非常明显的变化是，界面中的信息密度大大降低了，同时spec中多次强调留白的作用。Style这一章节非常细致的讲述了Color、Icons、Imagery、Typography的设计运用方法，也是首次将Imagery纳入spec。此外，5.0还引入了新的系统字体：Noto。

Components（组件）

　　一些组件在这一版本也有不小的变化，比如Dialogs的按钮在一般状态下为平面样式，表单输入框控件引入了浮动标签。更具体的说明可以直接阅读spec，更多细节就不在此一一列出了。

　　如此详细的Spec大大降低了开发者的设计门槛，遵循说明便可以设计出中规中矩甚至看起来还不错的App界面。48dp在spec中依旧是一个非常重要的尺寸，最小触摸响应区域沿用了这一尺寸。

出神

发表于 2014-11-13 | 分类于 随笔

偶尔看看电影听听音乐也能出神是件幸福的事，因为日常生活已经把我们搞的机械不堪。

最近看的两个电影是艾德伍德和绣春刀，虽然故事的原型不太一样，但是都体现了人性中求发达的那个侧面。

艾德伍德一直拼命想拍部叫座的电影，改善自己穷困的窘境。沈炼想要那几百两金子，以后去过好日子。艾德伍德没办法让自己的狗屁科幻片被人接受。沈炼也没办法改变自己和兄弟完成任务就要被做掉的宿命。

每个在格子间里卖命换钱的打工仔都有过出神的那么个瞬间–要是我发达了多好，不用再为谁卖命。之所以是出神。是因为这个想法大多不切实际，除非你像沈炼一样碰到个赌明天的机会，且赌注是生命，还不一定不蚀本。所以各种交织在一起你还是得入神工作。

电影里有太多这样的赌徒，速度与激情里的那帮人，美国往事里的那帮人。慢慢发型这些贼寇并不一定是为恶多端。而是要在一个阶级板结化的社会里重新洗牌更新命运现世投胎。古今一样，只不过赌徒的信念也许会在翻身之前垮塌，正如无数的平凡人生。

我以前很信命，觉得名人都是命运机缘使然。但是现在更多地看到名人其实也是很拼的。比如四处巡演的宋冬野。他们已然混的人五人六还各种熬夜奔波，你说你作为一个little nothing又有什么理由不去努力工作呢？也许每个人都不是那么喜欢自己的命运，但是你得学着去改变它。

幼儿园

发表于 2014-10-31 | 分类于 随笔

国庆回家的时候去了趟我的幼儿园。

我上幼儿园的时候，家就在幼儿园对面的小学。每天我妈最后一节课上课的时候就会让堂哥去接我。那时候门口有个小卖部的杂货摊，最稀奇的东西是五毛钱一盒的崭新蜡笔，但是记忆里零落的全是各种被用过和折断的蜡笔，因为盼着买一盒新的。那时候每天趴在幼儿园的水泥栏杆上写作业，跟小伙伴纠结的问题是写作业的时候用完橡皮要不要放回文具盒里——因为很快又会写错又要从盒子里把橡皮拿出来，我说一直放在外面吧，小伙伴说怕把橡皮搞丢。每天的日子也挺枯燥无聊，填不完的方字格，写不完的bpmf，总之是有好几百遍的作业。最盼望过六一儿童节和舞蹈比赛，哦，对了，那时候还不会吹泡泡糖，买泡泡糖就是用来嚼甜头的，直到咀嚼到没有味道就吐掉，有一次在音体室跳舞的时候跳舞一仰头不小心把嘴里没吐掉的泡泡糖咽了，这在那时的我看来很严重，一直担心泡泡糖把我肚子粘起来。

幼儿园装下了我所有4-6岁的记忆，当我再次走近，发现那些游乐设施一点没有变，可以钻过去的蘑菇桥还在，滑滑梯还在，跷跷板还在，他们还是他们，但是经过时间的剥蚀光鲜不再。记得那时候滑滑梯上面被我们的屁股蹭的贼亮到反光可以当镜子，栏杆上的油漆也还是滑溜溜的，现在他们都起皮了。是现在的小孩不稀奇这些玩具了么？也许吧，ipad和手机也许对他们更有吸引力。

音体室也破败的让我吃惊。那时候看到播放磁带的录音机觉得有一种朝圣的心情，舞台上的布幔也是酒红色的天鹅绒让人觉得神圣。音体室的下面刷着蓝绿色的漆，天窗可以看到外面的葱葱绿绿。现在音体室里面变得黑乎乎，很久没有更新过硬件的样子。然而门卫还是说了句让我吃惊的话，让你们这些家长进来考察考察看看条件不错就愿意让孩子来了。

我太傻了，我以为房子盖好了就永远不会漏雨。时间的力量太强大让这些物质产生又走向衰败。也许这些物质来到世界上的使命就是不断修葺自己不断更新自己，停滞不前的终会被淘汰掉。
