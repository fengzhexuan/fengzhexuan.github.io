正能量滚滚而来——《神探亨特张》观后

发表于 2012-07-24 | 分类于 随笔

　　上周六赶了个大早，去看《神探亨特张》，大约四成的上座率，但是看得人很嗨皮。

　　这个片子没有什么太清晰的主线，讲的只是那张坛子脸。

　　在老周的歌声飘忽响起的时候，忽然禁不住泪眼婆娑。

　　我有太多的负能量，张惠领的负能量是来自那些各行其是的小偷骗子，而我的负能量全部来自于我自己。

　　几个月来因为一些事情难以释怀，心放在刀尖，在睡眠不好的第二天悄然浮起，阵阵失落，无所适从。

　　这个社会险象环生，机关算尽不过是技术层面的尔虞我诈，也许我们都忘了出来闹腾的初衷——喷薄而出的正义感。

　　好莱坞盛产蝙蝠侠超人等英雄梦想，其实比英雄梦想更稀缺的是普通人对正义的无条件追随。

　　但是在中国特色的物质商品社会里，普通人也许不会那么在乎正义不正义。

　　如果真的有所谓的正能量，希望它滚滚而来，吞没你也吞没我，这样也许就不需要亨特张，每个人都是心灵强大的爱婶子。

在微博上写博客

发表于 2012-07-17 | 分类于 随笔

　　吃猪肝面腻着了，上来敲点字清爽清爽。

　　上微博上多了感觉很没意思，但是还一天天乐此不疲的当粉刷匠，回头一看好像博客的寿命天然要比微博的寿命长，起码博客是个有头有尾的家伙，微博就是一短平快的高潮，来如闪电去无踪影。

　　近几天看了一些人现在的过去的博客，让我愈发地坚信博客还会有第二春。

　　但现在人都还真不爱写博客了，我也是。

　　第一个接触的博客是msn space，那是05年底的事儿了，同学们纷纷入驻，敲文字贴图片粘音乐玩得不亦乐乎，更有甚者，去网吧通宵的时候，还在钻研别人的空间。那时候空间满足了我们的表现欲，激发了我们的创作欲，我甚至还学了一点点html语言，为的是贴出来不会显示八叉的图片。那个时候还喜欢小红莓，还喜欢故作深沉地吟出些感悟，此处省略一万字。

　　后来新浪博客来了，名人博客也开始出现了。同学中追徐静蕾的也冒出来了。为了激励自己学习ps，买了本ps教程，天天把教程学一遍，把学习成果展示在上面。当然没坚持了多久就变成了心情碎语集散地，再没多久就废掉了。我的同学还在坚持写，写到了零八年，还搬去了搜狐的博客，有的一直坚持在space，后来我也一直在space上有写，一个人写一个人看，写的很长很长，但是基调是灰暗。再后来space被关掉了，那些字也不知道消失去哪儿了。

　　其实我个人感觉新浪博客的功能是最简单好用的，只不过新浪老爱走名人路线，把老百姓吸引来看名人，但是名人又有神马好看的呢？娱乐圈明星就是秀照片，公知们就知道骂战，一开始我还看孔庆东的博客，后来觉得实在没劲。

　　对于博客的印象，比较深的就属韩寒和黄健翔，这俩人老爱在博客上发东西，2楼12楼22楼的盆友们有奖拿，吸引众多网民在文章后面盖楼。彼时热闹的博客，到现在剩下一幢幢烂尾楼，静静地矗立于07年到09年的某月某天。

　　我觉得博客的最大功能是记录，供以后翻看，微博基本上就是装满碎纸片的垃圾篓，想要把他们拼凑起来挂在墙上，还真挺难的。

交互设计师的特质

发表于 2012-07-02 | 分类于 随笔

　　用户体验在国内的互联网环境中基本上渗透率已经较高了，高校也逐渐将交互设计作为一个单独的专业来开设。那么，什么是交互设计师的特质？

好奇心
　　坦白的说，好奇心是一个“设计师”而不仅仅是“交互设计师”必备的特质。生活中的每样东西，不论好坏，几乎都是经过设计的。因此，一个合格的设计师需要充满好奇心：“它为什么是这样设计的，怎样设计会更好？”与此同时，好奇心是设计师的灵感来源。
　　举个简单的例子，国内几乎每个玻璃门上都标有“推”或“拉”的标识，尽管有些门推拉都可以打开。而较为通用的做法则是，装有横着的把手代表可以推，竖着的的把手代表可以拉，如此则较为符合一般人的心智模型。这里要吐槽一下公司的推拉门，每次进出都需要看把手旁边的标识是推还是拉。

同理心
　　要跟谁同理心呢？当然是用户。还记得白鸦和keso在UCD社区上有关“装不装用户”的辩论，我认为同理心是我们的目标，而装不装用户则是达到该目标的一种方法。作为设计师，有没有充分的了解用户的需求，而这才是同理心真正的核心。
　　就我个人而言，平常会接触到很多的产品，而在体验这些新奇产品的同时，却没有从用户的角度去看（有没有真正的去用），而更多的关注设计的亮点，这一点是以后需要弥补与加强的。

结构性思维
　　设计从了解需求到最后定稿，其中必然经过各种各样的设计，怎样将自己的设计理由表达出来是非常重要的。你可能会面临来自产品经理或其他团队成员的质疑，因此训练结构性的思维会让你的设计更为严谨。

　　以上，感谢青云的传授。
