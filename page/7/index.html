忘记了思考

发表于 2014-03-31 | 分类于 随笔

工作以后，如果没有大片的空余时间，很难有时间思考点啥。
每天下班以后顿觉一阵空虚，拎起包出门去坐公交车，盯着new balance574广告牌上的男男女女出会儿神，盼望的公交车来到，毫不费神地挤上车去。有座位的话继续支离破碎的思考旋即消化掉一天工作中的感悟和不快，赶紧把自己丢到菜市场去买菜。
变化一天天来到，终于到了要雕刻人性琢磨本质的时刻，我还是抵抗了，不愿意面目模糊地活。好似在疯狂地思考，却又好像什么都没有想起来，以后，将来，似乎总有办法，但也总有毫无办法的情况。
矫情好像少了一点，却总还想证明点存在感。孤零零，所以也不用去刷存在感，不会有恰当的回应。
渺远的，也想不出。当下，只能这样。

『向日葵』

发表于 2014-03-20 | 分类于 随笔

——你们走好了，
——向日葵下的孩子。

　　『向日葵』是网易云音乐每日歌曲推荐的一首曲子，一开始只觉得旋律有些忧伤而且好听，但隐隐觉得歌词背后有点故事。谷歌之后，原来这首歌收录在紫雨乐队于1998年1月发行的首张也是最后一张专辑『穷人的玫瑰』中，是为了歌唱1994年臭名昭著的克拉玛依大火中被「让领导先走」葬送火海的学生。专辑发行后这支乐队被崔健非常看好，然而就是这支歌让他们没了前路。

　　觉得歌词中「向日葵」另有所指，让人联想。找遍了几乎所有的音乐网站，这盘专辑音质普遍有些问题，几乎没有低音和高音的细节，像是从磁带中转录出来，但依然不能阻挡这首歌的魅力。

无病呻吟

发表于 2014-03-15 | 分类于 随笔

　　春节几乎让每个人都成了候鸟。

　　返乡日子的前20天要买火车票，抢票软件层出不穷，各互联网公司生怕不能在这杯强需求的羹里分一勺，让本来就疲于应对的12306更加雪上加霜。

　　经过一天一夜火车的颠簸，又踏上了家乡的土壤，到达省城。火车站外趴活的出租车司机从来不打表，本来到长途客运站20块的费用却生生翻了一倍。交警同志在几米外的警车里惬意地晒着冬日暖阳，好像失明一样。哦，对了，这个貌似不归他们管，不过摆设似的红绿灯在他们眼里也是陌生人。

　　到达汽车站，回县城的大巴发车时间尚早。找个地儿吃饭吧，不远处有家快餐店。走到点餐台前，只见人们把仅开的一个窗口团团围住，个个把攥着钱的手探到收银员面前，和着嘴里不断重复的菜名。

　　走了，乖乖地去买碗泡面吧。

家政服务

发表于 2014-03-05 | 分类于 随笔

转眼间已跟家政服务打了四回交道。
第一次：洗衣服的时候羽绒服上的毛毛掉到厨房下水管道，管道堵塞，call来了一个大叔，拿工具捯饬一通，揪出来那一条毛绒绒已黑乎乎的东西，不堵了。收费，五十块。
第二次：门开不了了。门锁本来就不好，有一次锁门的时候过分用力，于是人就被锁在里面了。没办法，再打电话，修理的小伙子到了先在门外喊话，修好了要二百。只好答应了，人家才开始动弹修。最后修好了跟人说：都是年轻人，便宜点吧，一百二。
第三次：水管断裂，半截在里面。又打电话，同一个大叔来了。先议价，一百五。没办法，答应了。把断了水龙头先取了出来，再装了一个新的。被坑一百五。
插曲：大叔安装的水龙头是淘宝上价格为个位数的水龙头，没用几天就开始滴水，滴的水不会从下水管道里流走，只会渗透到池子外面流到地板上一滩滩的。某一天男主实在受不了了，又tb了一个水龙头借来管道钳自己装好，这才解决问题。花费三十多。
第四回：也就是昨天，吃完晚饭洗了碗过了一会儿看到厨房池子里的水盆居然漂起来了，地上也一滩滩水。搞不清楚情况只得等男主下班回家。没办法，先找物业。物业关门了。取了点钱准备送出去解决问题。又call，同一大叔驾到，大叔确定问题是主管道堵了，所以脏水才往上冒，先开价二百，砍价到一百五。此时隔壁的邻居出现，说要是下次堵了咱们均摊，不一会儿快通了，邻居人闪了。在接了三四根管道通到一层（我们在三层）之后终于疏通。大叔说如此这般通管道也是有风险的，灭了我们砍价的念头。收钱，洗手，走人。邻居又过来聒噪一通，大意是以前他掏过，这次我先掏，也同时灭了我们向四五六层均摊费用的想法。
临了，大叔说：钱嘛，挣了不就是用来花的嘛，反正不会在你手里。
呵呵。

天注定命

发表于 2014-03-02 | 分类于 随笔

这电影还行。
因为首先，它算是一部真正的电影。
其实早就如先前的说法，这部电影真正的编剧是这个国家和这个时代。
观看这部电影的过程，其实就像在看网易的社会新闻。虽然不知何时起，总是屏蔽那些社会新闻，所以这部电影其实是逼迫观众去正视一些人本不愿意去正视的过去的事。人总是擅长遗忘，社会新闻被消费过后也一样被遗忘，但是社会新闻的背景都还在，你我他也许早就忘了时代和社会这两码事，继续若无其事点开韩剧和娱乐新闻，让轻松的愉快的内容驱逐那些无法思考无法作为的境地。
回到电影，四个故事都有种强烈的宿命感，大海必须开枪杀死村长，王宝强必须杀人抢劫四处流窜，赵涛必须跟苏三一样流落他乡，小辉必须跳楼消解自己和社会的隔阂。这些必须和不得不看似跟钱跟情有关，但其实他们只是幕后黑手掌中的棋子，操纵着一切的社会政治经济原因讲不清道不明，那就只能把这一切归诸于最大的“天”，换言之，这部电影演出来的只是各种苦果，没有揭示其中的因。
在影片中，几个农民坐在一个拉有耶稣画像的三轮车上，现实寓意不言自明。20世纪初义和团杀害外国传教士，一百年以后，中国底层老百姓开始抱耶稣的大腿。只是不知道外国传教士和中国农民都在祷告的那个耶稣是不是同一个。
假科长也许只是看到了历史和现实的惊人雷同，用电影手段拼贴了这样一个我们再熟悉不过的现实。可是又有谁会去思考呢？或者即使思考又有何用呢？一个不崇尚理性和光明的社会给行走在其中的人是不是只能提供片中那样的出路？
找不到原因就找不到出路，或者找到原因却还是只能听天由命，恰巧这一切都可以算账算到老天头上，老天受着这一切，但最终，还是你我他受着这一切，所以每个人都也是老天，都也是命运。
每每希望自己不要旁观，去正视去作为，但还是会被束缚住。所以我们都爱说“没办法”，却还是在幻想中努力意淫突围命运。
