如12306般冷酷——中国铁路客户添堵中心

发表于 2012-09-26 | 分类于 随笔

　　以为生活原本已经平静麻木蛋逼到没有任何感觉，

　　但是在压力之下还是会从某个罅隙里迸发出炽烈狂喜的火焰。

　　比如说，在买到火车票的那一刻！

　　以前，或者不久以前，在人人上看到吐槽12306的文章，总是会不以为然。自从铁道部开了打电话订票这个服务后，我就以为是大大的进步，并且于某年某月的暑假因为打电话抢到了某小镇回帝都的三张卧铺中的两张而窃窃沾沾自喜。随后使用12306购票的经历也还算愉快，先后顺利滴买了若干次火车票，所以我觉得吐槽12306是愤青的行为是庸俗的行为是未脱离低级趣味的行为，是在太平盛世中发出的不和谐的噪音。

　　但是，从今天早上七点到现在的经历让我对12306有了簇新的认识，在长达一个多小时与12306的拉锯战中，我终于拔得一筹票。

　　但是，我还是忍不住要喷出血和泪的控诉！

　　先人云，对待同志要像春天般温暖，今人说，对待敌人要如12306般冷酷。

　　以下是今天右撇子同志的私人工作日志：

　　7：00 闹钟作响

　　7：06 从迷糊中醒来，爬起来去开电脑，看火车票几点开卖，哦也，是八点，还有时间

　　7：09 喝了一杯酸奶，洗头发，洗脸，准备出门

　　7：45 杂事完毕，出门上班

　　8：07 到达单位，以最快速度打开电脑，还开始拨区号+95105105订票，结果电话打不通，12306倒是登上去了，看到还有11张硬卧，心中窃喜，赶紧插上优盾（结果随后悲催滴发现里面的钱不够了），战斗就此打响！

　　8:07-8:30 点击预定车票，但每次都是点了预订，订单成功，但是到支付的时候显示验证码不对，或者是已经点到了支付，但是又说没有你想买的座位！在反复折腾几次后，仅存的11张卧铺慢慢变成了无，靠，无奈又不甘心地打电话，确实是只有硬座！硬座！接着右撇子同学无奈又不甘心地继续刷12306，见证奇迹的时刻到了，系统的奶个部分居然出了两张票！两张！右撇子同学当仁不让地抢到了其中的一张。

　　8：30-9:19 在占到了茅坑以后，右撇子同学奇迹滴发现在经过几次购物活动后，卡上悲催滴只剩下了二百多块，二百多块！在给左撇子同学拨打了求救电话后，右撇子学还是悲天悯人滴担心到嘴的酱鸭子给飞了，于是乎右撇子同学厚着脸皮向诸位同事及其表弟借网上银行使，在连遭各种面带微笑内心叵测的拒绝之后，某位同学答应帮我先买票，右撇子同学飞一般滴冲回办公室，用同事的电脑登陆了12306网站（因为我的电脑要装各种插件），我在同事的电脑上输了一遍又一遍的用户名和密码，都被12306以当前用户过多为由拒绝，我的小肝儿绝望滴透心凉。另外，某位同事提供了信用卡可以买票的消息！这是个多么振奋人心的消息。右撇子同学激灵滴膝跳反应般滴想到了小刘er，小刘er乖乖滴把帐号密码都告诉了我，我乐哈哈滴开始在电脑上输一长串数字，还打各种电话催各种验证码。在经过了一番折腾以后，我距deadline还有十几分钟的时间，而就在这十几分钟暨我即将输入密码成功成仁之际，电脑学习了12306的三个戴表精神，神奇滴死机了！我勒个擦！但是，且慢！

　　9：19 就在我以为自己要与这张宝贵的回程票擦身而过时，我居然看到了已支付的订单！我在脑海中搜索着问题的答案，答案。。那就是。。你必须被艾特！左撇子同学！

　　综上所述，我终于买到了票，这对于一个已经调休了的上班人员来说是多么大的福利！

　　但是，我自觉我的吐槽还不够深刻，因为我还没有经历过传说中的排队支付。在对着那一行那一列的“无”字后，我愁苦交集，申冤无处，出头无日，真想变成个蠕虫钻进去黑了丫的。有人会说，没办法，这就是中国的国情；有人会说，没办法，他那个系统就是这么烂。我只能叹口气，因为我已经不再天真地幻想12306系统会有更专业的人士来操作更好的服务器来支持，我那点美好的愿望已经被一个老大哥制造的冷酷的网站给屏蔽掉，而我的呼声也只能消弭在互联网浩淼的比特中。

　　遗憾的是我不懂什么技术，无法写出技术帖以飨大家。在网络订票不靠谱时，确实应该打电话订票，但是在打电话也订不到票时，只能继续在网上刷新捡漏。所以说，今天我只是运气好，捡到了一张别人扔掉的票。在经过一上午的折腾后，我只能总结出一句很没营养的话，遇到困难只有坚持，才能看到希望。

　　所以我也只好变成了一个庸俗的愤青的未脱离低级趣味的被原来的我所鄙视的人。
　　我只能戏谑，但心中悲凉。

情感化搜索

发表于 2012-09-20 | 分类于 随笔

　　以前一直用豆瓣FM听歌，偶尔发现了Jing.fm。不同于前者的是，后者有一个大大的搜索框，可以搜索“温情”之类的词语或者该类词语与演唱者、乐器等的组合来筛选音乐。


　　这样的搜索可以将其称之为语义搜索（Semantic Search），而语义搜索正是Web 3.0中的一个概念。在介绍语义搜索之前先简单熟悉一下Web 3.0这一概念。简单来说，专家们将Web 3.0称作语义网（Semantic Web），语义的意思是数据驱动。网络收集用户的数据，使其从根本上满足用户的需要。
　　回到语义搜索，是指搜索引擎的工作不再拘泥于用户所输入请求语句的字面本身，而是透过现象看本质，准确地捕捉到用户所输入语句后面的真正意图，并以此来进行搜索，从而更准确地向用户返回最符合其需求的搜索结果。例如，如果你经常搜索“设计博客”，你将会收到很多与设计关联的广告。又或者，当你搜索“电脑”时，网络知道“哦，你经常搜索设计”，这样就会给出将“设计”与“电脑”结合的搜索结果。在Web 3.0设计中有更多关于它的介绍。
　　再看另外一个产品，Hoppit。相比Yelp，大众点评等解决的是哪里好吃哪里便宜的问题，而Hoppit则进一步解决了这样的问题：一家餐厅的氛围，是否温馨，是否安静，是否符合我现在的心情等等。


　　其实可以将这两个产品中的搜索称为“情感化搜索”，这种搜索同人与人之间的沟通更为相近，更为自然。像“温情”这种具备情感的词语，具备模糊、主观的特点，而支持语义搜索的产品则更像是人，“它”更懂你。这样的产品更像是从本质上遵循“情感化设计”，因为其本身已经成为了人。

众望所归——《蝙蝠侠：黑暗骑士崛起》

发表于 2012-09-03 | 分类于 随笔

　　周末去看了imax的蝙蝠侠，imax的效果确实不一般，虽然屏幕没有电影博物馆的屏幕那么老大，但是看到电影中各种摩天大楼的灯光星星般闪耀，还是觉得值了票价。

　　之前看过1和2，对2里面的小丑恨得牙痒痒，只是看了就忘，已经忘了小丑是怎么被干掉的了。所以在还没去电影院看之前，我就忧心忡忡蝙蝠侠会不会挂掉，提前去看了一眼剧透，啊，蝙蝠侠没有死，众望所归。

　　在3里面，贝尔的出场顿时把我拉回《金陵十三钗》，囧。纵观3的全片，贝尔是有点惨，但也不耽搁他穿上盔甲以后盘踞楼顶玉树临披风的耍帅。在这部片里，导演索性让福克斯给他造的顶尖武器都没排上用场，只拼贴身肉搏。在黑暗压垮哥谭的剧情进行到一半的时候，让人都很放心，反正他最后跟猫女去佛罗伦萨过上了笑傲江湖里任盈盈和令狐冲一样的生活。

　　贝尔的敌手看似是那个戴着头套声音诡谲的雇佣军贝恩，其实雇佣军也只是忍者大师的女儿的盘中菜。贝恩在布置好邪恶的黑天大网以后，得意洋洋地向市民宣布自己的计划，那场面有点像文革，谁有钱造谁的反，谁有文化造谁的反。那股子要消灭一切阶级和标签的劲儿不能不让人联想到mr.mao。他造反只是为了他挚爱的人的爱恨情仇，并不怀有“均贫富”的理想，就算是古代中国的起义军也打破不了“推翻皇帝自己做皇帝”的循环圈，那么贝恩也只是特别私人地在做这些事情。贯彻理想主义，太难。

　　里面的米兰达如何成为韦恩公司的董事？这一点编剧的痕迹有点重。猫女也是凭空地就想抹掉过去远走高飞，虽然猫女也曾怀有“劫富济穷”的梦想，但这样一个少年就越狱成功的女孩（左撇子注：猫女不正与米兰达遥相对应），不知道她的前因后果前世今生到底所思为何物？仅仅看着蝙蝠侠挨打就正义之心萌发么？

　　里面的警察布莱克反复地说到“制度是枷锁”，并在蝙蝠侠“舍生取义”之后将警徽扔入大海，也许制度和法律并不能保障正义凛凛。忍者大师一上来就叫嚣着要毁灭哥谭，所恶为何物？如果他憎恶的是三六九等的尊卑名分，那么毁掉这个城市真是没任何裨益。这背后的原因三个片子里讲的都不是很清楚。因为有恶，蝙蝠侠就要正义现身拯救平民？看似合理的情节编排之后，其实也很空洞。

　　倒是最后布鲁斯韦恩的墓碑表明蝙蝠侠确实要遁世做个隐士，与其口碑相传不如拥有幸福的个人生活。如果最后黑暗邪恶战胜了正义，大家都不能接受。但是全片里黑暗的情节远多于光明的乍现，也许这也正说明，正义其实是很难的，所以与其不顾死活地战斗，不如去普及一下正义的信念，就像影片里的罗宾最后找到蝙蝠侠的洞穴似乎要成为二代蝙蝠侠。用中国特色社会主义的修辞来说就是：“一个雷锋倒下去，千万个雷锋站起来。”

旅行的意义——《朝圣之路》观后

发表于 2012-08-31 | 分类于 随笔

　　朝圣之路，乍看给人苦行僧一路匍匐长跪到圣地的联想。其实它说的不是宗教，而是旅行，而是旅行的意义。

　　名人名言植入：旅行是消除无知和仇恨的最好方法。——马克吐温

　　在这部影片里，西班牙各种美景目不暇接。但是看到后来主人公酒后吐真言的那刻，影片的高潮来了，在日常的生活里，我们就像是在玩角色扮演，从来不会听听自己的真心。

　　这部片子虽然美景云云，但完全不像是一部旅游宣传推广片。它的台词、人物设定和情节完满地向你讲述旅行所能给人带来的改变。

　　“我要去环游世界”在众口相传中变成了一个俗不可耐的主题，但是如果有人真的去做了，它会带来比想的更多的改变。

　　片子里的男猪脚在上路之前完全没有踟蹰，也许，每个人都在等合适的时间出发上路，去寻找新的自己。
