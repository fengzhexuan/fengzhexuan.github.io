活着——《辛德勒的名单》观后

发表于 2011-07-02 | 分类于 随笔

　　老早以前就听说过《辛德勒的名单》这个电影，可能是初中的时候，语文报上的边边角角总是会介绍一些电影，比如《肖申克的救赎》。后者我看了至少两遍，一遍遍重温希望和不懈奋斗带给人的力量。但是前者不同，可能出于对残忍镜头与生俱来的恐惧，没有主动地想要观影的愿望。如果不是左撇子童鞋的钦点，我还没有机会接近这部“神作”。

　　这部电影带给我的震撼要更加强烈。可能暂时我们不会碰到纳粹灭绝某一人种的惨境，但是我们天天都在为更好的生活而奔波。这种奔波所需要的希望和力量绝不次于这部电影里犹太人所面临的灭顶之灾。

　　是的，生存的下一步是生活，而生活着努力着只是为了更好地生活。

　　这么活着我们要感谢的是自己。父母为了让孩子接受更好的教育而遭遇车祸，盲人们一天十二小时为恶人帮敛财，生活中的困境比比皆是，但我们如履薄冰地行走和兢兢业业的努力只是为了不再跌入更惨的境地。

　　《2012》里人类死命地奔跑只为逃脱末日来临的魔爪，即便没有方向。只是为了活着。

　　活着需要智慧，智慧需要积累。拥有智慧的前提是不要被SB们所蒙蔽，继续NB地寻找真理。

　　不管是什么，在世上活着得拿出种态度。

　　世界末日要来了，我们还是天天早起去种树。

　　寄生在时代大船上的我们，只有保持点信念，才不那么容易被淹没。接受这命运，像奴隶一样去努力。

侧边栏设计全解析（二）

发表于 2011-06-27 | 分类于 随笔


　　在侧边栏设计全解析（一）中，具体分析了三种侧边栏的优缺点以及适用情况。这次将分析侧边栏中应该放置一些什么内容。

侧边栏内容
　　这里应该放些什么样的特色内容呢？　　再次强调网站的定位
　　可能在网站的导航已经有类似“服务”的菜单，但是通过在侧边栏防止一些引导性的文字能够引导用户更深入的了解网站的定位、所提供的服务等等。　　放置最流行或最新发表的内容
　　可以把想要再次强调的内容发在这里，或者用不同的目录分类方法，需要略读的东西都可以放在这里。在这里的内容能够让来访者更快速的浏览，迅速找到感兴趣的内容。　　为社交增彩
　　侧边栏中通常还用来放置RSS Feeds、最新评论、Twitter Feeds、Flickr Feeds等。可以增大网站的影响范围，扩大受众。　　放置广告的绝佳位置
　　通常来说，广告嘛，当然要放置在显眼的位置，才能够获得其最大的价值。但对于来访者来说，他们是来寻找内容的，显眼的位置会分散他们的注意力。而侧边栏（尤其是右侧边栏），才是盛放广告的绝佳位置，虽然没有那么显眼，但在这个内容为王的时代，将广告放置在侧边栏已经成为一种通用的方法。
　　结论
　　网页结构对于信息的组织方式是非常重要的，尤其网页排版在借鉴平面排版的基础上。有效的利用侧边栏这块一亩三分地，能够给网站带来很大的益处。

那些关于自己的判断

发表于 2011-06-25 | 分类于 随笔

今天深深地觉得自己是个缺乏敬畏心的人，有时候有耐心地搜一首歌搜一下午，有时候却丝毫没耐性地摆着张臭脸。

有这个wordpress的博客，忽然心生自豪，因为以前看三表的博客，基于wp平台架构。现在有了，却没有像刚开始那样贡献自己寥寥又寥寥的文字，对左撇子同学心怀歉意和愧疚。不过这说明我还是个有责任感的人。哈哈。

现在很难逼着自己写点流水账写点记叙文了，一是觉得没劲，二是觉得自己表述生活的词汇越来越贫乏了。不像以前写东西的时候先亮出一个范儿，再努力把自己往那个范儿上套，削足适履吧？这其实也不太好。

也许不太那么重视表达，可能也是觉得生活中更需要的是解决问题的能力和行动，而非多么深刻多么全面多么独到的见解。如果你摊开书本，会读到这样的那样的诠释和隐喻，但是如果你去真正地生活，却像个小丑，那大概也很尴尬。所以，现在，读书于我而言可能更多的是个平静心情和打磨思维的过程。世界上的知识无穷无尽，而我们最迫切的只不过是过好自己的生活。

一句话，用行动去表达自己的感恩之心，比在这里敲字要来得爽。

侧边栏设计全解析（一）

发表于 2011-06-25 | 分类于 随笔


　　在网页设计中，页面的格局排版很多都借鉴于平面印刷的设计，两栏、三栏乃至多栏设计作为天然的内容信息的组织分类方式，在网页设计中应用的如此广泛。你的侧边栏会放些什么内容呢？本文就将侧边栏做个全方位的解析。

什么是侧边栏？
　　侧边栏不是主要的内容区域，通常不会作为导航的首选。侧边栏可以有很多实际且有用的目的。侧边栏通常是次生内容的容器，这些内容虽然不是主要的，但对于网站来说也是必不可少的。它们可以用不同于导航栏的方法来引导访问者，但又不至于将他们的焦点从主要内容分开。

左、右侧边栏
　　在网页设计中，首要内容是我们应该首先考虑的，那如果一个网站需要侧边栏，应该将其放在左边还是右边呢？如果有两个侧边栏的话，应该一边一个还是都放置在某一侧呢？虽然这些都是细小的问题，但对于网站来说却有不可忽视的影响。到底该如何放置，下面就各种情况的优劣进行分析。
　　左侧边栏

　　 优点：能够第一时间引起来访者的注意，在页面架构中处于更高的位置，所以适合用来做首要导航。比如，分类、文章标题等能够更快的引导来访者查看主要内容。如果首要导航的列表中内容太多，在一个水平导航中不能盛下，在这样的情况下，左侧边栏来替代导航是个不错的选择，比如说亚马逊、京东首页左侧的商品分类。
　　 缺点：因为其右侧的内容才是主要的，所以如果在左侧边栏呈现不合适内容则会误导来访者。通常来说，右侧边栏是常见的设计方式，所以如果左侧边栏没有使用特别的目的，会降低网站的可用性。
　　右侧边栏

　　 优点：通常来说，右侧边栏标准的设计方式。因为侧边栏中放置次要内容，也符合右侧方位在架构中处于稍低的位置。对于稍小的屏幕来说，如果不是自适应或者响应式的设计，则侧边栏不会完全显示（甚至完全看不到），这一点也是设计中需要注意的地方。
　　 缺点：右侧边栏处于次要的位置，如果将重要的内容放置在此，来访者将不会注意到。另外，广告通常处于右侧，尽管广告需要让人注意到，但是它毕竟是次要内容。
　　分裂侧边栏
　　一些网站有两列侧边栏，一边一个，也即所谓的三栏设计。对于需要呈现大量信息的网站来说，这不失为一种合适的方法。使用这种设计需要注意的是，将稍重要的内容放置在左侧，次要内容放置在右侧，所以，该种方式结合了左、右侧边栏的优点。

　　这次先介绍到这里，下期将就侧边栏应该呈现哪些内容进行深入分析，敬请期待！

LV的精彩

发表于 2011-06-21 | 分类于 随笔

　　说不上来对这个展感兴趣是什么原因，既不是它目前的潜在消费者，又不是个拜物主义者。也许只是因为潜意识里觉得这会是个精彩的展览，展览奢华，展览诱惑，展览一个品牌的传奇。

　　这个展览的规模算不上大，四个展厅中有两个是比较疏离于LV包包本身的：一个展厅是展望（人名）的行为艺术和雕塑，另一个则是LV的广告、秀和小小的数码互动。除此以外的两个展厅则是这次展览的精华。其一当然是从创始人开始，念念叨叨一通交通工具的演变——其实这正是LV包包诞生的理由——使用交通工具去旅行的路上，你需要个盥洗室、写字台、卧室、衣帽间、户外运动品、小冰箱等等，LV会以一个箱子的方式给你所有一切。纯手工是噱头，订制是借口，得瑟才是LV包包的实质——假若一个人在旅途中拎着LV包包，那么看到他这个包的人会刮目相看。即使旁人看不到你那包包里装的是牙刷或者鞋子或者衣服，LV包包都会倾倒一大片人的目光。所以带着你的LV包去旅行才是正经事，大开间小工位上的白领只会羡慕嫉妒恨你那价格不菲的LV包包。




　　其二则是LV的独家订制包包走廊。这是为各种“控”并且有钱的人打造的专属包包。放眼望去，有羽毛球控、雪茄控、洋娃娃控、小提琴控、摄影控、鞋子控、衣物控、帽子控等等等等，这些有钱的“控”们不忍心让自己心爱的洋娃娃独自呆在家，硬要让洋娃娃和自己一起上路，怎么办？LV帮你解决，让你的洋娃娃跟着你一起走南闯北，再回到你的家。其中村上隆设计的玛丽莲LV包，里面则摆满了十来个LV包，看来这是为LV控打造的。






　　这个展倒是消除了我对LV的一些误解——LV天然不是炫富的工具，只是在中国的国情里被扭曲成了奢侈品的象征。奢侈品其实包含两个方面，一方面是贵（财富的象征），另一方面则是生活的质感和品位。在中国，财富常有，而品位则不常有。LV包包在中国可能只能是盛放爱疯4、化妆品和TT的容器，拎着LV包包的人可能根本不知道这个包包为什么这么贵——换句话说，大部分享有LV包包的中国人只是拎着LV包包的机械手。

　　也许不应该责怪中国，毕竟这是个来不及获得内在谈论内在的时代。拥有一个LV还是大多数人的梦想，谈论内在更是梦中梦。待到人手一个LV时，我们是不是就开始重视内在了？

Web 3.0设计简介

发表于 2011-06-17 | 分类于 随笔


　　这是一个Web 2.0的时代，是信息爆炸、快速传播与分享的时代，是社交媒体的时代。看国外，从Youtube到Facebook，再到Twitter；看国内，从博客到人人网，再到微博。还记得Web 1.0的年代么？将信息放到网站上展示，通过邮件进行私人交流。互联网发展将近20年，Web 3.0已经呼之欲出，那么Web 3.0的世界将会什么样的呢？

Web 3.0是什么？
　　很难说清互联网如何发展出来像Facebook和Twitter这样的顶级网站。当你研究Web 3.0的时候，你会发现用户与网络的交互是注定要发生的事情。在Web 2.0中，我们将注意力集中在用户之间的交互或者交流上，现在我们将更加关注用户本身，这是永远不会错的。
　　专家将Web 3.0定义为语义网（Semantic Web），语义的意思是数据驱动。网络收集用户的数据，使其从根本上满足用户的需要。如果你经常搜索“设计博客”，你将会收到很多与设计关联的广告。又或者，当你搜索“电脑”，网络知道“哦，你经常搜索设计”，这样就会给出将“设计”与“电脑”结合的搜索结果。
　　Web 3.0的益处
　　Web 3.0最NB之处是你可以在任何地方获取数据，这得益于各种智能移动终端以及其背后的云计算。想想各大公司都对云计算这块蛋糕在虎视眈眈着，IBM、Amazon、Apple、Microsoft等。物联网也将对Web 3.0做出一些支撑，自我感觉虽然有交集但是服务对象仍然有差别。
　　Web 3.0对设计的影响
　　对于设计师来说，Web 3.0将会有什么影响呢？Web 2.0时代的设计建立在吸引、说服用户的基础上，如此，为了让用户能够在线小费做出了很多努力。你想买这个吗？那赶快点击这个按钮吧。这就是Web 2.0的设计准则。
　　众多的设计元素被添加到Web 2.0的设计中来，比如线性渐变是不可或缺的元素之一。不管你想要怎样的颜色组合，从背景到按钮都存在线性渐变。其他的设计趋势，比如各种徽章、圆角、图标。那么在3.0时代设计师该怎样做呢？
　　Web 2.0 设计 vs. Web 3.0 设计
　　如果说Web 2.0设计是为网络设计的话，那么Web 3.0不仅要考虑网络与浏览器，而要考虑到所有的媒体。随着智能手机与平板电脑的迅速普及，用户想要通过这些终端获取更多的数据、得到更多的服务与应用。现在很多网站为了满足不同的需求，同时维护桌面与移动终端的网站，如何让同一网站能够适应不同的环境则是Web 3.0的一大趋势，比如 响应式设计。
　　同时，让事情变得简单仍然将是设计师专注的目标之一。设计师完全有能力引导用户浏览页面的先后次序。不论何时，设计师要让用户关注其喜欢的内容，而不应分散其注意力。Logo与导航栏也同样是不可忽视的设计要点。
