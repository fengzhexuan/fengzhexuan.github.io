活法

发表于 2013-10-21 | 分类于 随笔

前一段时间看了「百万美元宝贝」，一个努力的女拳击手追逐名利梦想，但最后被黑拳打倒，最后生命与梦想两空的故事。看完几天以后，左童鞋还隐隐叨过一句：「为什么结局是这样？」
走着走着忽然碰到了一个选择：是用高强度的工作来换丰厚的余生养命钱？还是挣着薄薪庸庸碌碌过余生？
生命有限，它的子集–青春更加有限，鉴于很多饭碗是青春饭，所以快速聚财成为我们生活的主题，是富有地名扬四海地die young还是穷困潦倒中长寿离开平凡的世界？
命肯定比钱重要，但是活着没有钱，似乎生活也没有了什么质量和内容。
人，可以是工具，赚钱的工具。人，也可以是目的，生老病死时用钱延长生命的进度。
所以这一对矛盾真的如马克思辩证哲学一样在人身上对立统一。其实这种对立统一只是一种说法，没有提供人最终的活法。
我们可以从别人那里照存生活的方式，但永远没有现成的答案。
鲍勃涤纶大叔的话很万能百搭，the answer is blowing in the wind。起码他发问过，也许找到了答案，也许没有。起码，我也审视过，思考过了自己的境地。只能如此。

遥望

发表于 2013-10-14 | 分类于 随笔

又经历了一次从北到南的颠簸，身心俱疲，但脑子里装着的还是家乡的风景和记忆。
这次来了以后感觉诸多抗拒，那种漂泊的感觉愈加强烈，天高地阔但我只有自己的那一方牢笼，走在大街上好似进入无人之境，因为来来去去都是陌生人，而我，好像也不存在于这个空间了。
时间流淌终有告慰，这颗刚经历乡情乡音洗礼的心又能赤子多久？几天以后这种宁静也许就会被城市加之于身的事务所破坏，被通勤的吵闹所打乱，而彼时的我又要去哪里洗濯被弄脏的心情呢？
我们自觉不是现实的人功利的人，少不经事时还能对现实的户口房价和世俗嗤之以鼻不入心不入神，慢慢却也落入求稳求安定的俗套，我接纳我现在这颗正在变俗的心，也接纳彼时不羁的自己，年轻的时候谁没有傻过，晚熟是会吃亏，只是各自品尝人生的甘苦冷暖吧。
当我看到自己青春已然褪去，心境越来越甘平淡，不再渴望超越心中的标杆，也失去衡量成功失败的标杆，慢慢无所谓，自己想要的东西也慢慢清楚，当初走出来的地方才是让我能真正心安的地方，这是多么痛苦的领悟。
在遥远的地方，我在忙些与自己的心灵关系不大的事情，只有在思乡情切时，念念脑海中家乡的风景，空空默默遥望。

幽闭的媒体

发表于 2013-10-11 | 分类于 随笔

隐约记得王小波评价张爱玲的小说是幽闭的小说，人物兜兜转转总是那些命运。
现在很多的时间都在weibo，偶尔看看杂志也没什么新鲜信息，比如某杂志一篇写王菲的文章就是凑了很多旧时的八卦和网上的时令语言，作者下的功夫可以说是搜集资料翔实，也可以视作是一种毫无创造的拼贴–因为那些八卦n年前就看到过，这跟幽闭小说有一定相似之处。最近真正花功夫的王菲的文章在三年以前三联生活周刊上。
不看南周也很久了，主要那文章越来越像实习生写的，没有以前读南周那种打翻水桶信息量大的感觉。
现在能从南都娱乐周刊上读得些边角的猛料，其他的娱乐新闻真是泯然众人，而各种大小网络公司还有千百号的实习生亦或是员工点击全选复制粘贴把这种新闻转载到自己网站上，制造出数量庞大的平庸报道的链接，这更加幽闭。
想想也怪可怕，但这就是现实世界的一角，值得读的长微博越来越少，所以诺奖热也还有点警醒作用，有多少是文学，有多少是文字的垃圾。

反省

发表于 2013-10-04 | 分类于 随笔

回到了局促的小城镇才有时间有空间来反省一下自己的日常生活，那些每天在城市里早起晚归挤公交打不到车害怕迟到扣工资上课跟学生同事上级周旋的生活，在此时此刻想起来显得渺小不值一提。每天在做的那些事其实并不是生命中特别重要的事，每天在辛苦的事情只是为了口饭吃，而孤零零附着在城市表皮的我们却把他们当做全部。
在人群中就容易忘掉自己，每次总是回家以后脑子才能清静不为外界左右。比起疯狂上课赚钱，还有更重要的事，找到自己，做自己这个年龄该去考虑和行动的事情。
从前从来没有过要去创业的想法，现在愚钝不灵的我居然有了想趁年轻试试看的想法。一直朝九晚五只是一种活法要搭上生命中更多的时间做不重要的事，也许是时候去改变了。
＜每天叫醒你的不仅是闹钟，还有梦想＞，毕业一年多我的梦想早已难寻，每天都是在上班的通勤疲惫中睡去，在回笼觉没睡够的困意中起身开始一天。做了很多想的太少，好像已经满足苟且于现在的打工生活，在一天又一天的奔忙中梦想沉睡过去，是应该叫醒梦想了，如果它，还在。

表情经济学

发表于 2013-09-24 | 分类于 随笔


图片来自于网络

「表情贴纸」在社交网络中的需求有多大，为什么越来越多的公司通过卖表情来盈利？

　　首先来说说表情的作用：
　　表情从最初的符号（比如 orz、:-) 一类），逐渐发展出后来更为丰富多变的贴纸（静态、动态），更造就了兔斯基等经典表情形象。
　　现在，表情出现在互联网任何可以输入的地方。与人类发明的其他工具一样，表情自然是人类自己表情的延伸。在用文字符号不能满足情感表达时，表情的作用便开始体现。当然它的作用不仅限于此。

　　公司卖表情的目的：
　　如此看来，表情贴纸在社交产品中是强需求，尤其在更希望能够自由表达自我的年轻人群中。有需求的地方便可以有买卖。
　　一方面，在丰富产品的同时，用 freemium 的方式为公司带来盈利。
　　另一方面，用众包的方式来完善表情贴纸这一生态，毕竟公司拥有的设计资源有限，让广大的设计师通过分成等方式获得收入，为产品服务，实现作者、产品、用户的三赢。

　　所以，这是一件好事。
