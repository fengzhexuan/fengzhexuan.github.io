游戏人生

发表于 2014-07-02 | 分类于 随笔

去年看爸爸去哪儿的时候，一开始没啥感觉，只觉得kimi萌萌的，cindy好能跑，石头很懂事，天天么感觉不诚实，王诗龄就傻傻的。第一季的活动都很轻松，做饭为主，找食材为辅。等到第二季感觉几个爸爸过的很辛苦，也许是因为关注了黄磊和他老婆的微博，他们经常在上面po出来的经常是大人或者孩子在睡觉的照片。这年头做人不容易，想上节目出个名博个赞也是体力活，要被摆弄来摆弄去的。

吴镇宇也在微博上po过这么一条：太太帶兒子看完大上海，說兒子知道原來我是壞人很不不開心，我只可跟四歲的他講，爸演壞人是為了賺錢給你買玩具，誰知他說以後不買玩具，我也不能再演壞人！我說好！之後就開始擔心，如果某天演愛情戲，他又投訴我為何抱著的不是媽媽，我怎辦呢？不過他今天已改口說要玩具，爸爸去演壞人吧！太懂事了。

越来越觉得生活的苦，普通人是这样，明星看来也一样。生活越来越像一场游戏，打不完的关卡和小怪兽。

布达，佩斯，大饭店

发表于 2014-06-25 | 分类于 随笔

以前爱看电影，电脑里下满了各种片子，甚至把一个导演的电影都要撸一遍，也许是年少时爱看故事画报的后遗症。上班以后没有以前那么爱看电影了，费精力费时间，还得费时间出神。

看布达佩斯大饭店是因为天才瑞普利，在天才瑞普利中，裘德洛灿灿金发美人尖俊朗表惊为天人。后来上豆瓣一搜，一看他最近演的片儿就是大饭店了，其实大饭店也过去一年了吧，记得去年戛纳的时候有看到过介绍。

这个电影整个过程很轻松愉快，一部分原因是这个片子的布景虽然很假但是很讲究，没有像国内一些年代片布景一看就不属于那个时代。另一个原因，就是因为各种情况先急转直下，然后主人公在黑暗中摸摸索索情况又大好。剧中的演员在雪地里跑的很夸张，像舞台剧里的那种高抬腿又向前跃的那种脚步。

看的时候觉得这片子肯定是欧洲人尤其是法国人导演的，但没想到是一个美国导演导出来的“老欧洲”。

影片快结尾，什么东西都归国有，旧时代留下的物质遗产都消逝，只有往事留在zero心中。

邮局是一个特别让人感慨的地方，因为总是离别一个城市的时候，总是会到这里。
邮局里人来人往，而我注意的只是袋子上的地址。天南海北。
去年冬天，一个学生在九莲新村邮出了自己所有的行李，迫不及待地回了青岛。
今年夏天，一个在九莲庄打工的妇女本想邮寄一包被褥，邮政人员检查了行李，说超重不邮，还有，你的褥子棉絮这么破了还是别邮了。正当工作人员拿标价十元一个标准大袋子之际，妇女忽然说不邮了。
还有几个女学生不愿意换袋子，麻丝袋缝的也不好，工作人员说摔破了不负责，学生说没关系，工作人员说你都不对自己负责的。

人话

发表于 2014-06-20 | 分类于 随笔

最近在wb上看到一个漫画网站.
wb上那几幅倒是没太大感觉，只有下面这幅引发了强烈的共鸣。

先试着翻译一下：

A:小瘪三，侦察到么子东西？

B:我找到了一颗酷酷的石头诶

A:这跟我们的侦查任务没半毛钱关系。

B:呀，你看，这上面还有条纹儿呢，虽然这么看上去有点猥琐。

A:这跟我们的侦查任务没。。。（嘴被塞住）

B:去你妈的，会说人话么

忘了从几岁开始知道有沟通这个词的，这个词乍一听很愉快，经过谈话聊天想法畅达。但其实后来才知道有时候沟通是如此的艰难，先不提最后能不能达成共识，光就让信息到达另一个生物体的大脑，让那个大脑literally处理每字每句就已经很艰难——因为你不知道那个大脑之前接受过什么狗屁教育或者洗脑培训是不是还有读懂别人话的功能，更甚，也许它不是读不懂，而是懒得去懂，而拿出来一些先成的说辞来对付你。所以信息不是在一条线上线性进行，而是两条直线平行进行。

希望我们有运气多碰到点真正的人，说点人话。如果碰到异形，注意避让。

bye bye，富士屋

发表于 2014-06-18 | 分类于 随笔

第一次吃富士屋是在2010年春天，那时候还在跟着导师做项目，据导师说他们同学每周都省钱，等周末来吃一次56元的自助日料。第一次知道加州卷，北极贝，天妇罗，北极贝是大家的宠儿，蘸着芥末和酱油很好吃。具体的滋味已经忘掉，但是自助不限量吃的很开心。

第二次吃富士屋在2011年夏天，富士屋的烤鱿鱼是我吃过的最好吃的烤鱿鱼，但是那次吃三文鱼吃的很猛，刚出门肚子里就翻江倒海，吐掉了。

第三次吃富士屋在2011年冬天，左撇子去给eico design做调查问卷得了两百块，而且也得了饭碗，刚好吃一次日料庆祝一下。富士屋的牛舌也很不错，吃到嘴里有种跟牛的舌苔接吻的赶脚，但是盐椒入味特别好吃。除了刺身，日料现做，每次做好一盘门铃按一声，基本上每道上来以后很快就光盘，所以大部分时间是在执著等门铃响起。在这家基本上主食都是鸡肋，不如多点几只颜色鲜亮又柔韧的鱿鱼。

去那里的有blind date的男女，老少齐上阵的一家。装修得朴素，桌子上的小碟子不出意外总是装满鲜绿的芥末，没上菜的时候就嘴馋拿筷子蘸点芥末吃吃，饥饿的青春时代啊。

今天上网搜了一下大众点评，它居然已经关门了，虽然这也是意料之中的事。艺海大厦一楼，富士屋，再也不见。

一样的夏天

发表于 2014-06-15 | 分类于 随笔

年纪越大，时间过的越快，也许因为日子每一天都过的越来越雷同，所以穿梭其中越来越不需要任何想象和感觉。每天的routine都一样，如果说跟以前相比每天开始关心什么，那也许是天气吧，温度几何，晴雨是否，都很实际。再者就是季节，还会有点切肤的记忆。

前年的时候雨水很多，刚到南方对一切感到稀奇，清晨中雨的节奏配着“就算大雨让这座城市颠倒我会给你怀抱”的闹钟，还沾沾自喜。下雨很多很大，逼着我买了双雨鞋，穿上雨鞋去面试的路上因为鞋子不舒服弄了弄，结果把用了七年的天堂伞弄丢。八月的头上来了海葵，等台风的夜里像是在迎接重大事件，仿佛会有什么改变。但只是看到了白色的雨铺天盖地下到了第二天。那个夏天空调开很少，懵懵懂懂，麻木着就过去了。

经过了没有暖气的冬天，特别渴望夏天，渴望艳阳，晒干潮气，驱走阴寒。第二个夏天步履匆匆来到。去年本应该印象深刻，因为四十多度高温频频炙烤着城市和人群。每天都很挤的公交车，空调不开的话简直比桑拿房给力。每天奔向单位开启上课模式，晚上做梦也是课不间断的梦境。不知那时有什么劲头支撑着自己打了鸡血一样卖命。然而到七月底还是颓了，闷热的夜里特别想插翅飞回清凉的北方。八月是天天倒计暑假结束的节奏，温度太高太残忍，厨房很热外面很热，空调房里不合时宜的冷侵入肌骨也不舒服，席天幕地无处不是热的牢笼。在那个夏天没有想过自己很疲惫，不敢打算未来，只生生想着撑过眼下艰难的境地。

这个夏天，快要来了。蚊子来了，咬的夜里睡不好，缺觉的一天好似行尸走肉好像什么都跟自己无关。半夜醒来，抹清凉油才能止痒勉强进入将睡没睡的境地。清醒与梦境的边缘，好像摸到蟑螂从身上爬过，登时马上醒来有了意识警惕地盖好床单。在应该清醒的白天，以前关于夏天的记忆也在体内泛出，夏天有三四场离别，有那么些被烈日能量催化的白日梦想，在这个夏天统统已经让位于朝九晚五的谋生计划和现实筹算。

日子只剩下，夏天，空调，西瓜，冰激凌，睡个好觉，精神饱满地迎接下一个繁茂的相似的夏天。

移动倒逼桌面

发表于 2014-06-03 | 分类于 随笔

　　花一上午的时间看完了WWDC14的开场Keynote，对于各种新系统、新特性、新功能的介绍，IT媒体已经有很多文章，就不赘述了，在这儿说一点自己的看法。

　　最大的感受如题所述，移动倒逼桌面。

　　其实说倒逼有点夸张，融合可能更贴切一些。消费者每天面对越来越多不同种类的屏幕，PC、手机、平板，却希望能在不同的屏幕上消费相同的内容。苹果自iCloud起已经希望把一些关键信息通过云端共享解决，今天苹果更进一步：在PC上接电话，iMessage消息同步显示等等。让同时拥有Mac OS、iOS设备的用户具备更高的生产力，当不同设备之间可以无缝连接、管理内容，想想还有点小激动呢。

　　在设计中，移动也对桌面产生了一些影响，这个另开新篇，再叙。

两棵桑树

发表于 2014-06-01 | 分类于 随笔

小学的时候，从同学那里弄来一只蚕。每天看着它白色软糯的身体在硬纸盒里凹凸来去。好开心。

过了几天，好像意识到了点什么，蚕也是要吃东西的。蚕吃桑叶和蒲公英叶子，城里没桑树，于是每天放了学直奔地里疯魔去找蒲公英。

后来从学校到家里路上的蒲公英都被我采光了，周末就跑到外婆家求她给我找桑叶。

外婆家梁后有一片桑园，但也总不好老去跟人说我去拽点你们家桑树叶子，虽然人家总会让我姥姥摘点桑葚给我们吃。

再后来表哥都去找发小寻桑树叶子了。每次拿来一塑料包的嫩绿新叶，带回城里够蚕吃一个礼拜。

蚕疯狂地啮噬桑叶也长肥了，成熟了，结茧了，死掉了。第二年看到黑色的种子似的黑点，但也没有蚕宝宝降临了。

后来表哥不知从哪里弄来两棵桑树，种在外婆家院子里。两棵树现在已经长得手腕粗了，如果养蚕的话，再也不用发愁桑叶了。

今年听说外婆催各个亲戚去采桑葚，说你们再不来就被鸟吃完了。我的水杯里，也泡着桑叶茶，很清甜。
