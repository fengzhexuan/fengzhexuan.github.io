昌平

发表于 2013-05-08 | 分类于 随笔

第一次知道昌平是因为海子，说海子在政法大学教书，那时候还不知道昌平是北京的一个郊县，总觉得关于北京的一切都很美好。
后来07年真的去了北京，住在昌平，觉得离市区真他妈远，那时候十三号线还不是很挤。
再后来去昌平是10年了，那时候满腹心事，也忘了昌平是个荒凉的地方。
再再后来，11年的冬天真的很冷，从四惠坐上753长途颠簸到昌平，人数稀少的公交车上，空荡荡的冷。
那个冬天还有次失眠，大早上去德胜门坐车去昌平参加考试，教室的桌子上落满了灰，找工作进入了痛苦的阶段，所以还是冷。
昌平离繁华有点远，离现实很近，有卖烤面筋的小摊和台球厅。
还有山西的面。
从前做的梦，以前走过的地方，深深地在心里埋着了，在某个时刻想起了，然后又忘记了。走走停停想想，路就走出来了。

灵隐好

发表于 2013-04-23 | 分类于 随笔

今天又去了一次灵隐的大门前，上次是去年了。
时间过的好快，今天是阴天伴着零碎的雨点，是个狗天气也是好天气。
上午的时候到了少年宫附近，那种老旧的湿润让我感到扑面而来的熟悉。
北山路那边的湖平静的像一条条黑白灰的线，空气氤氲着空濛，远处的山被白色袅娜地掩住。
断桥的石头被雨水洗刷了以后那种颜色黄的很亮，游人如梭，时间也是。
湖边的柳树嫩嫩地抽枝，在风中荡漾，让人心情也轻轻地摇摆。
坐着公交车看着路边的密林，感觉好熟悉，晴天雨天我都来过的。
这一年我也走过了，只是现在那经历过的一切事情和心情都变得模糊。
灵隐还是那么安静，空山石径的绿意，旧旧房子的星巴克，这一切都配得上美好这两个字来形容。
渐晚的天空微蓝，路上安安静静地只有我一个人的脚步，公交车站里的路灯照得树格外绿，看着砂地上自己的影子感觉好满足，也许这种心境就是日常生活里最高级的追逐。
灵隐好。

爬山日记之狮子峰

发表于 2013-04-09 | 分类于 随笔

昨天有友人邀约一起去爬山。去了上天竺。
一开始我就悲催地发现相机没有装记忆卡，所以只能拿着眼睛干看。
那一带茶园很多，我们却坐过站了，所以又从梅岭隧道往回走，隧道里车辆呼啸而过在拱形隧道里呜咽成鬼哭狼嚎，那种气氛让我想起了电影。
出了隧道走到了上天竺寺庙，从旁边岔开的小道走去。便开始一段攀爬。
山还是蛮陡的。
翻过去确实是好风景。梯田上的茶树密密麻麻，像一段段挤出来的牙膏，间或有戴着斗笠的女人们在采茶叶。作为北方人看到这些还是吃惊的。
顺着茶园往下走便是龙井村，走一段出去就搭上公交车到黄龙洞了。
春天去爬山，真的好开心。
