灵隐

发表于 2012-06-17 | 分类于 随笔

　　今天去了灵隐寺。

　　出发的路上一直在犹豫，因为下着不大不小的雨。等车时公交车急速刹车溅起的水花，让人急忙闪躲。

　　公交车窗外是一片接一片安静的绿，路上也不是很堵，很快就到了景区。

　　先买了飞来峰的景区票，这是进入灵隐寺的第一关卡，到第二关的时候仍旧要买票。

　　雨一直在下，游人大概是比平常要少一些。

　　看到飞来峰上雕刻的佛像还是吃了一惊，因为北方来的土鳖只在博物馆里见过石头刻成的佛像。山脚下掀起一点点洞穴，从这边穿到那边，中间是吓人的黑暗，有着窒息到亡的感觉。

　　从飞来峰紧走几步就到了灵隐寺，售票员收了人民币甩出两张票，急忙收好奔向入口。

　　灵隐寺不大，但是从入口一看里面的建筑物和物件都很大气，大殿都是一致的雄伟，佛像都是一致的高大，还有那椭圆的大香炉里不徐不疾燃着的热浪，把香炉顶上的雨水都氤氲成了水汽，慢慢腾空，消失。

　　先是弥勒佛，再是释迦摩尼，这些佛虽然身躯伟岸，但都微微颔首微笑，跟我以前见过的板儿直的佛像不同。

　　外面雨水冲刷着绿叶，庙内传来诵经的声音，领唱的一丝不苟，后面的和尚不知为何都没有剃光头发，草草咂咂的头发从各个脑瓜上冒出，认真又好像不认真地对着经书看着，翻着。最后一排坐着的照例是些穿着缁衣上了年纪的妇女，和着这诵经的声音静坐着。

　　一个大殿后面的照壁上刻有心经，字字句句念过去，有了些或深或浅的领悟，身在凡世中，谁能达到其中所说的“无意识界”呢？

　　一个一个大殿拜过去，便往回折返，最欣赏的还是那大殿里的高柱上所挂的牌匾，一个长难句便将佛的旨意传达清楚，印象最深的莫过于“感化”二字，心生善念感人化己，不知这算不算佛教的方法论。

　　从灵隐出来便继续在飞来峰周围闲逛，拾级而上，两边都是茂密幽深的树林，在这林子里埋伏的寺庙很清凉也很寂寥，从庙里出来路上还碰到了一只肥硕的蜗牛。

　　雨越下越大，只好准备撤了。

　　来了灵隐寺，才知道北方寺庙里化缘的“功德箱”到南方改名为“金香”。

　　到了山里才知道别有一番天地，静谧，只是我们不曾触及。

　　神灵隐居在山里，今天我们上路去看了看他。

那些地铁站的清晨与黄昏

发表于 2012-06-03 | 分类于 随笔

礼拜一到礼拜五，实验室蹲，礼拜六礼拜天，是outing的季节。

因为礼拜六礼拜天的出行，就有了他或者她在地铁站等人的时光。

是在西单，是在张自忠路，是在建国门……

是在寻觅美食的途上，是在去往郊外的地铁上，是在互相探望的漫漫路上……

轨道上的列车发出各种怪叫，匆忙的人流进进出出，报站和喊话的声音络绎不绝，但他们仍在等待。

忘了那时是怎样的状态，有时已经到了目的地的地铁站，但感觉特别的疲惫，于是会临时改换行程。

就那么几条线路，来来回回，翻来覆去却又不会厌烦。

那些地铁站的清晨与黄昏，将会是记忆中闪亮的日子。

想象力和创造力

发表于 2012-05-07 | 分类于 随笔


　　知乎在今天做了寻找中国互联网思想者的活动（似乎是个不错的运营活动），马化腾提出了这样的一个问题：
整个人类处于互联网发展的哪个阶段？下一个十年，互联网升级的大致方向在哪里？
　　得票最多的答案是这样写的（尽管它被折叠了）：
真是个很糟糕的问题，预测未来和判断方向本来就是扯淡的事情，想不通怎么还有这么多人假充专家来回答，仅仅是因为这是马化腾先生提出的问题么？”
　　未折叠最多得票答案可在问题页面中找到，很长。
　　相比quora中相似的问题：
互联网在2020年会是什么样的？
　　得票最多的答案是这样的（翻译版）：
想象这样一个网络：
　　互联网将无所不在，任何人将在任何地方通过任何设备接入互联网。它不是一个你去的地点，而是存在与你身后的一个层面。
　　通过更自然的用户界面交互。轻触一下你的手表，一个屏幕将展现在你的面前。想象《少数派报告》中的屏幕还有手势，没有鼠标，没有键盘。
　　无时无刻提供给你个性化的定制信息。比如，商场的屏幕能够认出你并展示给你重要的邮件。或者轻触屏幕告诉你是否要继续观看《House》第四十季，或者帮你转台到湖人队主场迎战凯尔特人的比赛。
　　直接连接大脑。指令将不仅仅通过NUI（自然用户界面），还能直接通过你的想法。想打开你的吸尘器吗？在脑海中想一想吧。可能这样的实现将需要很长的时间。
　　让你创建和构造几乎所有的东西。使用Web，在几秒钟内创建一个完整的虚拟世界。用它来指挥工厂中的机器人，来制造你的梦想之车。
　　连接我们彼此。语音通信有一天会被认为是过时的。甚至情绪会通过网络共享。
　　以上的一切将仅仅是个开始……
　　相比来看，Quora中的答案即生动又易于理解，不是说国外的月亮有多圆云云，这的确是我们需要不断学习之处。想象力决定创造力，在这个想象力被广泛抹杀的国度，还有很长的路要走。当然连接想象力和创造力之间的还有执行力^_^。

假作真时真亦假——《黄金大劫案》观后

发表于 2012-04-28 | 分类于 随笔

　　本来对国产电影少有期待，宁浩除外。

　　怀着满心期待去看了宁浩的新片《黄金大劫案》之后，结果却不尽然是如约的欢笑和泪水。

　　（1）特效太多。

　　（2）前半部分情节比较合情合理，后半部分开始无视正常情理地发展。

　　（3）笑料具有春晚特色。

　　（4）方舟子的质疑：@方舟子王水是浓硝酸和浓盐酸1:3的混合，很不稳定，会释放出大量的氯气，都是现配现用，不能运输，不能储存，没有成品。如果有人配了一油罐车的王水要去溶解黄金，周围的人都得被氯气毒死。更不要说油罐车先要被溶解掉。

　　（5）感觉有点像让子弹飞，却离姜文的叙事水平还有点距离。

　　（6）台词功力下降。

　　（7）主角之一也是新人之一表演腔浓重。

　　二流导演的三流作品，这个毁人的时代和大环境啊，只能期待贾樟柯了，他的严密话语体系会不会也被这个碎片的时代解构掉？
