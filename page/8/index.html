「无人区」观后

发表于 2013-12-16 | 分类于 随笔

无人区，没那么好。
完全利用了「被禁」这个噱头进行炒作，一个电影要多无奈才会使用自己被咔咔过这个点来为自己叫卖。宁浩，说延期上映这是「囤票房」。看在宁浩都说自己是三流导演的份儿上，我就不说他自我感觉良好了。
回到电影本身，说实在的还是对宁浩有比较高的期待，期待他能驾驭多样化的剧本讲不同的故事。但是，无论是黄金大劫案还是无人区都延续以往疯狂的石头系列的故事原型，三拨人为了一个东西打打杀杀。
在剧本里面，也有重复使用的手段，比如用车子撞人撞房子。
无人区的场景是粗犷的，但是编剧是有点弱的，下次宁浩再与这个编剧合作，大概是不用去电影院看了，去年就被这个黄某编剧坑过一次，这是第二次了。
大陆的电影真的不用再期待了么，贺岁片不准备去电影院了。

那时花开

发表于 2013-11-14 | 分类于 随笔


　　昨天终于看到了老狼的现场，了平生一桩心愿。

　　他的嗓子还比较靠谱，现场表演水平也比较老道。看到他脸上的肉开始往下掉，在台上打拍子的时候有点老太太扭秧歌的范儿，毕竟岁月不饶人。

　　九年前十年前我还在读高中，那时候也不会有很多的演出信息，上网都还很稀罕。更不要想去看一场演出。看老狼现场的愿望大约是上大学的时候，在听完老狼两张专辑之后，有想法的。那时候听「晴朗」觉得特别好。

　　这专辑是高二出来的，记得我们那个高中每个教室都有一个用班费（太久远的名词，跟红旗本一样）买的用来播放英语听力磁带的录音机，有一个女生买了这个专辑的磁带，在教室里用录音机播放，当时的欣赏水平只能欣赏得了周杰伦和朴树，当时买朴树的专辑，送了一张朴树的海报，我爸根本不认识这人，却还把这海报认真地拿图钉定在墙上，扯远了呵呵。拿录音机放「晴朗」这专辑，我跟那女生讲这专辑不好听，她很有矫情地说你运气不好这是b面，a面还不错。但是到了大学感觉一听都很顺耳，时间和心情际遇各种使然。高中太傻，也基本上很封闭，只听得懂流行歌曲。

　　那时候因为最好的朋友买了一张「青春无悔」的专辑，我借来听了听觉得还不错，就也去贡院街买了一盘磁带。那时候不了解高晓松也不了解老狼和叶蓓。只是偶尔从「萌芽」上看到b小调雨后等等。那时候还有女同学会做文学梦，写作文很好上个校报觉得很卓越。「青春无悔」听了很久很久直到高中毕业。所以里面的歌曲特别熟。

　　那时候在教室的投影上看了高晓松的电影「那时花开」，他的电影实验性质太浓，高晓松真不是个好导演，可是那时候丝毫没有判断力，觉得我怎么可以连人家的电影都看不懂，朴树周迅夏雨的搭配可以说是青春逼人，但说实在的，电影讲的事情太扯淡，包括他后来搞的其他电影也是。

　　那时总觉得音乐文学很神圣，看了安妮宝贝能郁闷很久，金锁记也是那时候看的。可是到现在那些在文化沙漠里视若珍宝的东西现在好像都成了前朝往事。老狼在台上调侃要返场演出要加钱，我们越来越实用，越来越无梦可做，除了赚钱无事可做。

　　那时花开，花开花谢，奔天涯。

习惯的力量（续）

发表于 2013-11-12 | 分类于 随笔

　　半年前的今天，有感于产品改版对使用者习惯所带来的改变，查阅资料并整理出习惯的力量一文。在这半年里，随着产品进一步的改版，对这一问题也进行了更多的思考，遂有此续章。

改版VS温水煮青蛙
　　提到著名的改版案例，不得不提Path在2011年11月发布的Ver2.0。这个完全重新设计的版本，使得Timeline和内容发布这一核心功能更加凸显，让人眼前一亮的设计更让产品在舆论网络上广为传播。

　　在更多的时候，产品的更新迭代更像是温水煮青蛙，偏向于稳健。譬如，iOS从发布至今的各个版本，系统的核心交互几乎没有变化，最大的改变非iOS7莫属。

　　在个人看来，iOS7整体上的改变有两个方面：一是以期更好服务内容的视觉表现，这是使用者能直观感受到的；二是趋于纵深的操作系统结构层次，这样的改变使用者不能非常直观的感知。而Spotlight的功能入口算是功能层面比较大的改动，所以升级到最新的iOS7时便把这点告知了使用者。Spotlight改为可以在主屏幕界面向下滑动随时唤起，比之前需要一直滑动到最左方便许多。不过需要吐槽的是，Spotlight首次唤起时依然会有卡顿。

　　再来看看来回折腾的Windows 8。Windows操作系统其实也一直在温水煮青蛙，尤其是著名的开始菜单在使用者的心目中完全无法动摇。但Windows 8却把开始菜单的图标隐藏，相应的操作还在。不过在最近发布更新的Windows 8.1中，开始菜单图标又回来了。

　　改版要注意的地方，总结起来只有两句话： 要改的比原来好，改了最好告诉我。

习惯&传统
　　当一个人的习惯变成一群人的习惯时，便成了传统。譬如，古时中文的竖写版式源自竹简时代的写法；又譬如，端午吃粽子中秋吃月饼重阳要登高。所以，从某种意义上来说，宗教信仰也是习惯使然的传统。

　　在交互设计中也有这样的传统。比如网站Logo的作用是回到首页，且一般在网页的左上角的位置，当使用者形成这样统一的认知后，这一交互行为也就成为了传统。不过也有例外，阿拉伯国家网站的Logo通常位于页面的右上角，它遵从了另外一种传统：阿拉伯语的阅读顺序为从右向左。

习惯在营销中的作用
　　免费试用是很多产品在初期甚至中后期采用的营销手段，不论是线下实体产品抑或互联网线上产品。当使用者的免费时段到期时，发现已经离不开该产品且不能找到相应的替代品，就不得不去付费继续使用了。充分利用了习惯对个人的影响。

　　最后，推荐一款App和一本书。App叫做 Habit List，如果想要培养习惯可以用它，这款App的图标非常有趣，生动的解释了习惯形成的轨迹。书呢跟本文同名 《习惯的力量》，从生理以及心理层面解读了习惯的养成以及要怎样培养或者改变习惯。

窦唯和他的「殃金咒」

发表于 2013-11-07 | 分类于 随笔

早上拜听了窦大神的新专辑-殃金咒，果然是重口味，却很不错。
很难讲这张专辑在讲什么，歌词基本上是各种嘶吼（有一句是「我这辈子跟你没完」），和着节奏的像念咒一样的碎碎念，粗重的男人的嗓音和配乐混在一起，偶来乐器煞一个尾声。这些乱七八糟的东西混在一起居然创造出了一种情境，抓狂配上节奏，这一种节奏反复然后加速，像一个手举菜刀过头顶一路狂奔一路尖叫的疯子，窦唯就坐在一旁看着这人杀过来再一次杀过来，嘴里只在念经以期安慰，尖叫消失，这种念经的声音逐渐扩大，警车（救护车）警报的声音响起，好似给这一切做了个了结，然后是吹气的声音，一盏灯灭。
第一遍听的时候感觉特别吵，感觉还是喜欢那些抒情小曲比如早春的雨伞。一向不喜欢嘈杂音乐，这次居然随着那些节奏听完了，那些无以名状的声响就像是日常生活里无处宣泄的躁动与不安，窦唯以这些节拍为主导率领大队的不安向你扑来，拥抱你一会儿，然后以一两声宁静收尾。
窦唯似乎还是忘不了古朴，那些金石之音，像古装恐怖片的配乐，但又不过时，像是现世生活里的音符。那些古琴笛子乍一听是前朝旧音，但又很时髦。
你刚觉得他要宁静，一大波群魔乱舞的噪音又已经袭来，这次的尖叫是歇斯底里的「吃了吗？」，两个机器人一样的声音开始对话，随后跟上一段曲。
在这里面，感觉窦唯是要刻意扭曲人声，不想叫别人听出来他要讲的内容，这些声音高远而宏大，好像是村里大队用来广播的喇叭坏了，又好像是自己在大声地对着镜子开导自己。无论尖叫的声音如何惧怖，窦唯的声音保持淡定安然，似乎看透一切的阐述自己的见解，像密密麻麻的咒语。
然后又是一段清丽的旋律，又是混乱的唢呐鼓声，但说真的，比那些在草莓摩登天空迷笛音乐节上的古琴琵琶搞的民乐摇滚强。个中原因，也许只能说是天赋和才华有差。
之前郑钧曾在博客里说「如今清高如小窦者只能每天在酒吧为三五个观众演出。你愿意这样吗？」随后郑钧说「我做不到，也不想做」所以郑钧现在跟着恒大音乐满中国跑，也没出新专辑。
窦唯就是穷清高吧，以真性情真刀真枪对付世人，却也在自己的专业领域苦苦耕耘成果累累，总之就是有才但情商比较低，不迎合媒体弄巡演上新闻出名挣不到钱，但心里有自己的追求，不向世俗低头。
各有各的活法，窦唯骑自行车带着小女友出行，每年整点高质量音乐，挺好。他也许没有很多物质，但我们也不需要太多物质，有点精气神有个乐观的态度，就够了。

火车纪事

发表于 2013-10-27 | 分类于 随笔

火车是这些年最多用的出行方式。从家到学校，每次都是坐火车。火车上也有许多故事。
有一次过完年去学校，对面卧铺上有个去北京做家政的中年妇女，一路上跟伙伴聊得很高兴，顺便还展示了要去带给雇主的土特产品–腌的荠菜。
没想到，半夜一个短信打乱了她的计划–她伺候的老人过世了，也就意味着她这趟去了北京要另谋他路了。
她有些失落，大概那个老人对她还不错，待遇也不错，在外打工碰到善人不易。她的伙伴安慰她说自己在通州有认识的人在打工的地方也要招人。她叹了口气，说这老人这时候不对，怎么不能再早一点，她还可以有打算的时间，她对这些毫无准备。
第二天，到站了。在抵达北京的那刻每个人都又好像充满了气的娃娃，眼睛亮了，溶化在城市的人海里。好像一切都没有发生过。
