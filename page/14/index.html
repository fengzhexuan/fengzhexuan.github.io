90%90%90%90%90%=不及格

90%90%90%90%90% = 59.049%=不及格

十年，非典，张国荣

发表于 2013-04-01 | 分类于 随笔

十年前我还不认识张国荣的时候，广东人和果子狸让我认识了非典。
非典来了，学校都放假了，记得那时候还下雨。老师不知道跑到哪里去了，我们在教室里大声说话等着放假。
早上的时候看了一个短片。
非典10周年重发视频纪念

从这个片子里有幸看到了十年前的北京，空无一人的天安门广场，游人稀少的故宫，还有戴着口罩的各色人等，对着额头可以量出体温的先进仪器。
那时候传说喝板蓝根有用，学校的食堂里还有中药。
放假了还有人从学校捎来月考的卷子。
我认识这一切，但是我还不认识张国荣，也不认识抑郁症，更不知道同性恋，也不知道唐鹤德。

05年的冬天在班里的联欢会上我终于知道了张国荣的一首歌。后来在网吧里看过霸王别姬，听到那句我本是男儿身又不是女姣娥后莫名泪如雨下。
总之我对张国荣的认识很有限，也不懂为什么那么多人喜欢他，也许是因为得不到的看不到的总是最好的、

我觉得80后很多都经历过非典，对于一场瘟疫也许人们没必要去纪念它，但是我们好像鸵鸟一样，对于自己屌丝的处境讳莫如深，而对远方的高富帅怀以诚挚动人的怀念。

回到那个短片，里面不出所料的有爱大婶。他是中国最好的纪录片制作者。张国荣已无处追寻，但爱大婶的片子总能给你些在中国活出人模狗样的力量。

娄烨你不懂爱——《浮城谜事》观后

发表于 2013-03-23 | 分类于 随笔

　　从《颐和园》开始就不喜欢娄烨，虽然他能拿一些这个那个奖，但是一个得奖的电影看了以后让人抑郁，在我眼里不是个好电影。

　　娄烨的电影以性和政治为噱头，但是性在娄烨的电影直接的反映就是ML，除了ML无他，所以娄烨之前在微博上愤怒抗议总局对他电影的审查。但其实术业有专攻，娄烨大抵是坳不过日本动作片导演的。

　　看看从颐和园到浮城谜事，男男女女在床上的表情，基本上是倾泻兽欲的狰狞，只有做没有爱。

　　但是无法不去直视娄烨在叙事上的长足进步，浮城谜事很紧凑很虐心，围剿追击小四的道路上正室和小三都很用力，但观众就只能看着郝蕾的怨妇表情憋着憋着，那个饰演小三的齐溪演技不错，看到最后才出了口气。还不如去看昆丁的cult片脑袋被踩的喷血匕首插入胸膛转轮手枪喷出子弹毙命的壮烈来的大快人心。

　　从戏剧元素来看，娄烨的电影缺乏佳构剧中的denouement，有了结局但是难以让人得出logical conclusion，不具有故事的原型，也无法纳入电影叙事体系。重视过程的铺张，但是缺乏逻辑上的收敛，让人郁闷。

　　浮城谜事中的小三事迹虽然确有其事，但是so what?这种悲剧的来源是什么？该往何处去？娄烨向来不擅长给出一个说法，看他的电影就像走在大街上的一个陌生人拿走你口袋里的五十块钱却不给理由一样的蛮横，美其名曰导演电影，大概就是导演想怎么样就怎么样吧。

　　总之，娄烨，喜欢不起来。

说冯唐

发表于 2013-03-18 | 分类于 随笔

　　大概前年的时候知道有冯唐这个人的存在，读得第一本他的书是万物生长，这是他最好看的一部小说。

　　后来读了他那本很火的你二不二，觉得大失所望，境界并没有那么开阔，不如红拂夜奔。

　　再后来在图书馆读到了他的杂文，能与时俱进，有自己思考，带着点儿拜物教的精神，比小说好那么点。

　　单向街还在蓝色港湾的时候去辗转公交看过他，金丝眼镜，文邹邹，很看得起他粉丝的智商。大概那时候很为工作焦灼，所以记得他说看他书的人至少是可以谋得生活的，所以很愿意相信他说的话。当时录的音很是得意，但到现在全然记不得他说的话了。

　　前不久看他的三十六大，他杂文水平有长进，但有一毛病就是在说喝酒的时候会提到他箱子里的呕吐袋，在说他箱子的时候又会说到那次大酒喝伤了，不知道这是互文还是语言贫乏亦或是个人生活经验不丰富导致想象力贫乏。但这本书确实挠人想象，不知道钱多到那么多的人怎么还会有精神生活。

　　中间还穿插他给李银河写的书序，那是一个文学青年装逼露鸡巴的调侃之作，与李银河没半毛钱关系。还有李银河编纂的他那个怪物文集，不咋地。

　　但我还是略微欣赏他，会去看他的微博。那些只言片语的幽默感或者我辈难以企及的物质与精神生活，除了装逼。

　　所以这是一个粉丝的批判性思考，多少文艺女青年欲求其精子而不得，而我亢奋的年代早已经过去了，注目深藏在心里的角落。
