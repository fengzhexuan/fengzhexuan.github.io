关于浮动下拉菜单的交互问题

发表于 2011-11-04 | 分类于 随笔

　　如果想要在有限的空间内显示较多的内容，那么浮动下拉菜单是个不错的选择。通过将隐藏内容在鼠标滑过菜单时显示，这是源于桌面软件菜单选项的一种设计方法。


　　但是，笔者在使用58同城的过程中却发现了该种设计方法的一个小问题，而且只是在某种设计方式下的独有问题。下面是58同城顶端“导航”的浮动下拉菜单，你可能会说这种设计不是挺正常的么？那就让笔者细细道来。


　　或许大家还记得迪士尼游乐园的那个故事，当所有建筑都已完成后，设计师为道路的设计烦恼起来，最后全部铺满了草地，让游人来选择出最优的道路。根据两点之间线段最短的原则来讲，以58同城的下拉菜单为例，当用户需要从导航移动到房产的链接选项时，哦，你是不是已经知道有什么问题了？嗯，是的，你会发现菜单收回去了。下面这张图说明了问题所在。当用户想点击房产链接时不得不走个折现或者弧线才能实现其目的。


　　如何解决这种问题呢？
　　1.使用左侧对齐的设计方式，但有时不得不使用右侧对齐，所以这不是一种十全的方法。
　　2.使用谷歌下拉菜单的交互方式，点击显示，点击浏览器窗口任何地点隐藏，虽然保持了一致性原则，但使用户操作数量增加。


　　3.时间延迟，当菜单失去焦点后延迟零点二三秒地时间，让用户有合适的时间度过危险地带，顺利抵达目的地。这是笔者目前觉得比较合理的解决方法，如果各位还有更好的解决方法，请不吝赐教。

通过追踪错误来提升用户体验

发表于 2011-10-16 | 分类于 随笔


　　提升用户体验是个细活，会涉及到产品或者服务的方方面面。一个互联网的产品，通过使用诸如Google Analytics之类的工具可以很容易的跟踪到浏览最多的网页、导航问题或者转化率等等。但是这些数据并没有反映出用户在使用过程中的绊脚石。跟踪并优化错误信息将能帮你显著提升用户体验。　　

要跟踪什么？

　　表单错误与404错误是最值得跟踪的错误信息。表单错误可通过客户端或者服务器端的表单验证进行捕捉，当然错误发生时你也可以触发Google Analytics时间进行跟踪（本文中都以Google Analytics为例进行说明）。　　表单错误

有关贾樟柯

发表于 2011-10-05 | 分类于 随笔

　　有太多次跟贾樟柯擦肩而过，从很早就知道的《小武》和《站台》，到他来s大的讲演，前者在硬盘里躺了好多年无人问津，后者压根没有去凑热闹。

　　为什么忽然又开始对他感兴趣呢？可能是探寻一些在外漂泊的关于生活和生存的答案，我以为在他的电影里可以找到。但是看了《世界》以后，只是看到他对彼时现实的成功描摹，有思考，但是没有答案。

　　看了《小武》以后，突然意识到所谓的大片大部分是超现实主义的巨制——《满城尽带黄金甲》《无极》《三枪拍案惊奇》，是脱离我们日常生活经验的，也不像一些好莱坞大片的传奇故事或者英雄梦想。

　　说实在的，他的电影很闷，灰头土脸的，不像华丽丽的狗血剧总是灯光璀璨人面桃花，但总能在国外的一些电影节上拿奖。起初以为是外国人的猎奇心理，把发生在大陆的一切现实的影像当做异域的奇观——就像我们看拉斯维加斯这座赌城的眼神，像是好奇并且准备下结论的窥视。但在观影后，我发现不是这样的。贾的电影方法和第五代的逻辑有着太大的不同，尽管都是剪切尽管都是在中国拍的，他的电影让我们看到了同样是中国然而发生在《新闻联播》之外的影像故事——不总是那几个部委的白底黑字大门牌不总是胡宝宝温贝贝亲切地抓着群众演员的手，不总是神州同贺九州新春黄河奔腾长江入海流……说实在的，那些新闻里的事情很多我们都难有共鸣，更多让我们感同身受的是到处拥挤的人流和商店里播放的老鼠爱大米两只蝴蝶。

　　贾樟柯自觉或者不自觉地担当起了记录这个时代的重任，像文献一样记录着大时代之下的个体的心绪——在闷罐车里被挤得没正型的张三李四王五会国骂一句。

　　他的电影在国内始终面临着尴尬的票房，甚至被第五代不屑甚至攻击。这也反映了我们这个时代多么娱乐至上多么消费主义的境况，被洗脑了的人们只愿意没事偷着乐，谁还愿意正视那么魔幻的房价和那么疯狂的现实社会？所有人背着身子扛着钱袋子走过这个时代，不愿意多瞧一眼，多说一句。也许是感到无力，毕竟个人对现实的关注是太渺小了。

　　贾樟柯自己也坦言：在追逐成功的过程中怕失掉自己的根本——低成本制作代表的关注现实的精神。我也担心在这么容易被碎片消食的时代里，他还能不能继续为描摹我们时代的民族群像贡献点力量？

　　期待他的新作《在清朝》。

　　3.明晚和女友打竞技场，第一次打竞技场没经验，求PVP攻略？

  • 问：听说第一次容易秒退……问下有经验的同志怎么解决！

  • 答：1V1吧？别害怕，很少有人第一次1V1就能打好
    　　首先要专心，把背景程序都关了。竞技场上短兵相接，血肉相搏，你的眼里只能有对手，不可以有杂念
    　　如果女朋友也是头一回竞技场，你要主动出击。如果她不是第一次，你可以稍微示一下弱，看她能不能带带你
    　　可以放点轻音乐，烧点精油，点个小蜡烛什么的，加加buff
    　　脱护甲要慢，充分放松。胸甲比较难解开，你可以先找个没人用的练习练习
    　　开战之后切忌急攻，一定要观察对手的反应，如果那个流血效果DoT太痛，可能会影响下一次
    　　第一次最好不要尝试任何偏门的战术。那些教学录像，尤其是国外的，传授的战术都比较高级，要有很强的操控能力和耐力，不说别的，那稳定的高APM你就做不到
    　　起手要有礼貌，要温柔，不能冲锋，也不要拦截，要有武德，不能像瘪三打排位赛
    　　记得，嘲讽在PVP里毫无用处
    　　如果你不了解对手的套路，不要轻易动粗
    　　即便对手希望你越粗鲁越好，也不要在护甲盖不到的部位留下战斗痕迹，人家明天弄不好还要去上课
    　　觉得自己不行了可以停止动作，跑一下位
    　　记 得 戴 护 具
    　　交战过程中可以适当放放真言术：爱
    　　适当地战斗咆哮
    　　打完别看过分数就登出，两个人抱在一起好好总结一下哪里做得好，哪里做得不好，互相鼓励一下

    　　如果实在是怕秒退，可以在上场前几个小时自己找个精英怪的竞技录像单练一次，把软泥怪先清了，然后慢慢等它刷。刚刷的软泥怪没那么容易跑出来。

    　　随着角斗双方经验的提升，竞技场的乐趣会慢慢增加，祝你幸福。

　　4.华西村是不是个骗局？

  • 问：或者说，有哪些公众不知道的见不得人的东西？因为怎么看都不对劲，类似 “too good to be true.“

  • 答：中国有几个市场经济下的集体经济试点农村。当地政府都当成典型来树立，旁边的村子可能还过着穷日子，这就是中国的现实。

    　　理论上，有个人说不管白猫黑猫，能被女人喜欢就是好猫。所以华西村也好，中关村也好，都没什么问题。但是华西村是典型的树立的典型，很多政策资源都向它这里倾斜，就像全国的资源都倾斜到北京一样。有北京这人各种资源富余的浪费的省份，也有像贵州、青海这样相对落后的地方。

    　　我去过齐齐哈尔的一个村子，这个村子的GDP每年11亿。相当于一大堆明星制造的中国电影票房的十分之一。但是这个村子的周边村子还很落后。但不管齐齐哈尔还是黑龙江省，都把这个村子当成往脸上贴金的典型，银行的大量贷款让村子一下富裕起来了。别的村子想贷款，没那个条件。这个村子成功的秘诀是——封建家长制加上计划经济加上市场经济。

    　　搜狗:

　　6.用户体验是不是被我们过度放大了？

  • 问：用户体验一再一再被各个职位的人提起。
    　　好像不重视用户体验就不配做产品、运营甚至技术。
    　　可是，我们心心念念在乎的用户体验真的能被用户体验到么？
    　　尤其是面对中国网民现状，内容是否依然重要于所谓的体验？
    　　天涯的UI并不见得美观。4399有很炫的设计么？
    　　我们这些“互联网人士”追求的极简、设计感，真的是用户喜欢的么？
    　　用户会因为一个按钮的位置、几个色差，而接受/放弃一个网站么？

  • 答：看什么阶段啦。
    　　1. 当你没钱的时候，你穿一身破衣服，能保暖就行，这是基本需求（可用性）；
    　　2. 当你有些钱了，你就希望穿一身好看衣服，这样能显得整个人更好看，这就是基本体验了（易用性）；
    　　3. 当你有闲钱了，你就开始买一些简单的首饰，涂脂抹粉画眼线，这就是增强体验（好用性）；
    　　4. 当你非常有钱了，你开始穿金戴银，貂皮大衣，这时候你要彰显的是品位，这就是品牌需求了（品牌价值）

  • 答：做个比喻吧。
    　　当你在一次旅行时走到一片生机盎然的草原上，看到一棵百米高的参天大树，你会由衷感慨这一生命的奇迹。你会联想到这棵树为了长到这么大的规模，必然经历了很多磨难，克服了很多问题，并且战胜了很多竞争的同类。你有很多感慨，而且你发现这片生机盎然的草原上不止是有这几颗树，还有很多其他的品种，于是更多的感慨油然而生……
