新站点SEO应该做的事情

发表于 2011-04-13 | 分类于 随笔

　　尼龙扣-Nilock建起来快有一个月了，在这期间我也看了一些关于站点SEO方面的东西，现在将一些东西总结在此，希望能给想要建站的童鞋提供一些帮助。

不要用图片或者Flash来建立整个网站

　　目前的搜索引擎还不能对图片中的文字进行索引，而且严重依赖Flash的站点对搜索引擎不友好。这样不是说就要放弃图片或者Flash，因为如果没有这些一个站点会显得相当无趣。相反，试着将一些按钮、工具条的背景都放在一张图片里（image sprites），这样 还会提高网页的加载速度。
　　许多人喜欢用flash来做二级下拉菜单，以实现一些华丽的动画效果。如果用javascript或者jquery来代替flash，搜索引擎会对所有div中的文字进行索引，所以不妨试试jquery吧。
　　另外需要注意的是，越来越多的站点已经开始升级换代他们的站点，是时候使用html5与css3了。
　　关注meta信息与站点描述
　　一个站点meta信息与站点描述会显示在搜索结果中，这些信息要吸引人们点击进入。所以，怎样描述站点很重要。
　　在SEO策略与可用性之间寻找平衡点
　　SEO与可用性是一个站点都不可或缺的两方面。
　　寻找这两者之间的平衡点，对于不同类型的网站是不一样的。如果你的站点是政府网站，那可能要求网站简单易用。如果是一个IT公司的站点，可能网站更偏向于技术前沿与流行趋势。当然可用性会有其他方面的优势。
　　浮肿和凌乱的内容会给用户非常糟糕的感觉，这样常常导致站点的跳出率很高。
　　同时使用XML和HTML网站地图

　　网站地图对于爬行蜘蛛来说非常给力。它们引导爬行蜘蛛对站点的每个网页进行索引。但为什么同时使用两种地图呢？HTML版本是为了站点的访问者，而XML则是为了爬行蜘蛛。

《几度飘零——大陆赴台文人沉浮录》观后

发表于 2011-04-12 | 分类于 随笔

　　这本书的名字起的很诱人，但是古兄编纂的这本书称不上是善本。

　　胡适、梁实秋、林语堂算是在建国以后从大陆绝迹的文人杰出代表，他们的沉浮在书中也是占着前面的位置。

　　本以为这本书会很难写，因为其中涉及这些文人的政治主张等。但是作者把本书的冲突集中到了这些文人和蒋氏政权的冲突上，算是给本书的顺利出版打了包票。

　　这本书中熟识的文人不多。在赴台以后，胡适成了中央研究院的院长，梁实秋继续翻译匠和教书匠的生涯，林语堂则广为著述文章。其他的一些文人的事迹则以批判鲁迅为主，令人震惊的是鲁迅的大弟子居然是被特务乱砍杀死的。文人投笔从戎，大概还是要小心。没想到从小读鲁迅文章的我们，尔后会看到鲁迅被其他文人讥讽为“御用文人”，真是可悲。

　　本以为这本书的资料会很翔实，文人的沉浮之路毕现。但是实际上这本书特别像是八卦小报上的文字，如柏杨那一章，主题内容是柏杨和五个女人的故事，让人忍俊不禁。

　　仔细想来，这本书唯一的亮点是，在谈到胡适的时候，作者写道：当大陆“大鸣大放”时，北京曾派人向在美国的胡适带话：“我们尊重胡先生的人格，我们反对的不过是胡先生的思想。”胡适听后大笑，说：“没有胡适的思想，就没有胡适。”的确，一旦把胡适的思想批倒批臭，其人格又怎么能叫人尊重？

　　总得说来，文人（知识分子）是精神和人格较脆弱的一个群体。他们中许多人或许没有参政议政的愿望，有的只是贯彻自己学识的主张。想要实现自己抱负，许多文人不得不投靠某一政权。

　　那些留在大陆的文人，也未必预见到七十年代的事，若是能够预见，他们还会留下来吗？人生的选择和际遇，真是让人感慨。

有趣的CSS3多重背景

发表于 2011-04-11 | 分类于 随笔

　　多重背景是CSS3中的新属性。顾名思义，一个元素的背景可以有多个且可重叠显示。利用这一属性可以实现一些有趣的效果，今天就向大家介绍一个。

　　在此之前，首先向大家介绍一个小知识，一个生物学问题。

cicada principle金蝉脱壳原则
　　cicada principle，姑且译为金蝉脱壳原则（感谢 右撇子童鞋）。蝉，这种司空见惯的小生物，每隔7年、11年或者13年（根据不同的种类）便会大量成群的出现，然后找到伴侣ml，在不久之后便会死去。这是为什么呢？原来，蝉的天敌鸟类、蜘蛛、黄蜂、蛇等天敌大多是2至6年的繁荣萧条周期。而7、11、13都是素数，减少与其天敌繁荣周期的重叠，在这样的周期里进行后代的繁衍能增加其存活几率。如果蝉的周期为12的话，那岂不是很悲催？
金蝉脱壳原则很NB，但是与网页设计有什么关系？
　　周所周知，利用元素的background属性可以很容易的制作出大面积重复背景，但有时候这样重复背景却显得单调不够丰满。这时候，我们就用到了cicada principle。下面先看这个例子。

　　我们用到三个半透明正方体png图片：
29-a.png 37-a.png 53-a.png

html {
background-image: url(29-a.png),url(37-a.png), url(53-a.png);
padding:0;
margin:0;
height: 100%;
}

　　如你所看到的，三种图像通过重叠覆盖又生成了新的颜色，且在很短的范围内不会重复，那会是多少呢？29px × 37px × 53px= 56,869px!而我们所用到的所有素材加起来的大小只不过7kb，相当神奇的效果。

　　顺着这个思路，我们还可以生成其他不重复的大面积背景，比如草坪、幕布等等，大家尽情的发挥想象吧。当然css3的多重背景属性只有当前最新的浏览器支持Firefox 4, Chrome 10, IE9, Opera 11与Safari 5。

　　想看演示效果请猛击这里 demo。

《单身男女》观后

发表于 2011-04-11 | 分类于 随笔

　　将近两个小时的电影，让人忍不住有想往后拖的冲动，好在还是按捺住冲动，拧着给看完了。

　　主角的职业身份都比较闪耀：投资分析师、投资总监、建筑师。注定的三角恋爱在经历了三年之后会有怎样的变化呢？

　　两个男人都在写字楼宇的玻璃里拿便利贴贴着笑脸，玩着游戏，为了一个女人的欢心。

　　看到最后，成了十分简单的二选一游戏。两个男人争啊争抢啊抢，女人好似天鹅飞呀飞，最后谁能叼到？

　　每次古天乐在即将得手的时候都会精虫上脑，在其他女人的大胸面前喷涌鼻血。高圆圆再怎么爱他，也无法接受这样的男人，纵使他为自己买来豪车海景别墅，纵使他在英语字典的marry那一页夹上求婚戒指，纵使他最后关头仍然没有放弃攻势……

　　高圆圆还是选择了吴彦祖，因为时机，因为他可以带来的安心。

　　所以越往后看越纠结，眼看着古天乐就要得手，杜琪峰导演还是顺应我意，让吴彦祖得逞~

　　说实在的，这个电影真的很普通。让我想起了《全城热恋》，女主角也是在用便利贴整出桃心的形状。

　　说点更实在的，《非诚勿扰》其实也是一个靠时机选择恋爱对象的电视游戏。

　　你以为自己有选择权，其实你的选择样本库已经被缩小到二十四个单身女生和五个男嘉宾。

　　再回头说到电影，这个电影的英文名是don’t go breaking my heart.从题目就能看出是让人心安的男人获胜，这个男人在电影中便是吴彦祖。

　　吴彦祖和古天乐在最近成了黄金搭档，《窃听风云》《枪王之王》，都有他们的身影。然而剧情越来越狗血，拼贴电影的元素越来越贫乏，甚至这个电影的插曲也居然是陶喆的《爱很简单》和王菲的《我愿意》。

　　不过还是会有许多喜欢吴彦祖喜欢古天乐喜欢高圆圆的人去电影院看这部电影，明星无疑是吸引票房的春药。但是明星演出的这部戏就其观赏效果来说却只是一副催眠药。

春天里的感受力

发表于 2011-04-10 | 分类于 随笔

　　春天，我们的感官对于世界是张开的，因而感受力也会变得敏锐。

　　春天，蓝天白云绿树红花，会让你油然而生一段好心情。

　　贴一篇旧文，留住美好的春天。

　　春天都快过去了，也没留下点啥文字，怪遗憾，补上！

　　年年都看见操场边上的一排不知名的大树微微露出点黄绿，在微风里安静的等待。经历过好几场栉风沐雨以后，它才姗姗的泛了绿波。不知道为什么，看见那树顶上擦着蓝天的绿就想起了草原，可能已经成为了一种直觉的感应，可能那无垠的绿海的源头就在这几棵树先绿的枝桠里。不几天，绿的更加张胆，丛丛簇簇地现出活力。总觉得柳树徒有报春使者的虚名，总是等到操场那边的树鲜亮的绿了，才能看见柳树慢吞吞的抽芽报到，现在还不是柳絮来的时候，我只是蓦然想起那些粘人的花，得等到五六月了。

　　春天，百花盛开，不错，先开是玉兰，羞答答的在枝头娇藏着容颜，等到完全开了也就快凋落，盅满则溢，也许就是这个道理了。迎春花则不管不顾的盛开，金黄耀眼像星星，又像顽皮的孩子，在一个树条上要挤得得满满才罢休，迎春树算不得好树，可是花一开就吸引人的眼球——灿烂，灼目。可能它是知道自己只有这一季的风华才不出尽风头不罢休吧？桃花和丁香开的比较晚，在春雨的滋润下，它们慵懒的伸了下腰，就欢欢喜喜的露出脑袋捧出花骨朵了。桃花夭夭，灼灼其华，现代人已经习惯了桃花的繁盛，可是它带来的会惊艳更多，试着走近一棵桃树，不，不是如锦的花，锦是丝织品，你看那桃瓣重重叠叠挤挤攘攘地有了毛绒的质地。丁香花开的让人感动，微小的都放不满一指甲盖的花，努力的缀满枝头，也争一点春光。白的纯净，粉的艳丽，紫的淡雅…

天津两日游记

发表于 2011-04-07 | 分类于 随笔

　　清明有一个小长假，当然不能放过它。这次，我们去了天津。

　　到达火车站以后，我们首先坐上公交车去往天大（公交车站就在火车站的旁边，好多路车都可以到八里台，在那下车便是天大）。当时校内的42斋招待所已满，所以我们只得寻了一处天津中医药大学旁边的宾馆——天中宾馆住下。

　　一切安顿好了以后，就开始天津之旅咯。

第一站：南市食品街
　　没来以前以为这会是个古色古香的露天小吃街，但到了才发现，原来是个室内的食品街。里面有海鲜、甜食等各种小吃。但是这些小吃好像已经丢了那份淳朴的味道。

第二站：鼓楼大街&古文化街
　　南市食品街邻近鼓楼和古文化街，这两个地方也是幻想中的“神圣”之地，但去了才发现并不是那么回事。鼓楼有点像新修的，古文化街除了泥人张和杨柳青年画社都是些卖旅游纪念品的小店，以及一些稀奇古怪的庙会小玩意。

第三站：海河公园&天津之眼摩天轮
　　从古文化街往东走便是海河沿线，走过了几座大桥没想到就看到了天津眼。彼时天色昏沉，华灯初上，海河中的游轮划出的潮流泛着金光，摩天轮缓缓着转着。夜色静谧，河边是散步消闲的市民，好不惬意。

第四站：五大道和周围的建筑
　　从宾馆到五大道的路上，出租车司机给我们介绍五大道是个租界和故居集中的地方。但是下了车以后发现那些故居全都转换了模样。即使是历久弥坚的建筑也被修善者涂上了“古旧”的颜色。

里弄

旧民居

第五站：滨江道
　　从五大道乘坐观光2路便可以到达滨江道，说实在的，滨江道这样的地方可能到处都会有——鳞次栉比的商场，各种青少年服饰旗舰店和熙熙攘攘的人群。还有旁边小巷子里价格不菲的天津小吃，都弥漫着浓厚的商业气息。而滨江道附近的里弄却保持着原来的风貌，那种商业的嘈杂似乎根本不会打扰到那些居住其中的居民，这一点让人很奇怪。新的旧的建筑互相辉映，让人不得不唏嘘:北京已经拆的什么都没有了。

第六站：西开教堂
　　在天津，教堂可能俯拾即是。但这个西开教堂确实值得一去。坐落在滨江道尽头的西开教堂已经矗立了一个世纪，彩色的外表让人看不出它的沧桑往事，但是教堂的内部确实让人称奇。一排排的椅子上绑有海绵垫，桌子的抽屉中还放有布道书——也可以说，虔诚的教徒让这座教堂充满了宗教感，希冀更多的灵魂被上帝拯救。

西开教堂外景

西开教堂内部

　　以上说的只有一些旅游景点，也顺便给大家介绍一下去天津游玩的其他方面

　　食：以超市的方便食品为主。因为当地的饭菜不很便宜，所以只好去超市买了一些熟食和面包充饥。

　　住：在天大附近有不少旅馆，这次住的天中宾馆属于中等水平的旅馆。

　　行：公交车很难等而且很挤很贵，所以打的也是个不错的选择。最好备上一份天津地图噢，可以让你省掉不必花的钱，步行抵达目的地。

　　总得说来，天津是一个让人感到温馨的城市。相比北京的匆忙，天津要淡定很多。晚上八点多的时候街上已经人很少了。操着招牌天津话的天津人都很热情地会给你指路。这次去的时候没有去天津外国语看欧式建筑很是遗憾，下次再去吧。

《我们俩》OST听后

发表于 2011-04-02 | 分类于 随笔

　　戏剧学院的穷学生租住胡同里老奶奶的房子，就有了这个两个人的故事。

　　后来才知道这个电影的原声是窦唯创作的。

　　古琴的悠扬让你回到胡同那些个安静的清晨或者傍晚。

　　现在能回想起来的镜头就是小马帮老太太收拾房子，把画给换了，那个阴森森的镜头不知为什么到现在还记得起。

　　现在的东四附近的胡同，都变得很热闹，但不再是从前那个电影里或者窦唯家住的那些胡同。

　　在鼓楼附近，还可以看到穿着虎皮花纹腈纶睡衣的男人穿着拖鞋去公厕解决问题。

　　在北新桥附近，像是供销社的私人水果摊，像是国营的理发店，新修的四合院工地堆积的椽子，在雨中玩沙子的小孩子。这一切在雾中仿似一个独立的世界，不似灯红酒绿的那个北京。

　　建筑也是维系人情感的一种纽带，过去的建筑便沉积着我们关于过去关于自己来历的记忆。

　　胡同也是。

　　有点希望它是《卡拉是条狗》里面的那些个胡同，或者《百花深处》里面的那些已被拆迁早就当然无存的胡同——却满满填充了人们关于胡同的美好记忆。

　　前门一带本是胡同的世界，但改造以后的前门就像塑料花一样，虽不衰败但无法让给人带来感情上的舒适。正月十五，那幅人挤人的景象真是人堆的山和海。

　　前门一带，曾经有梁实秋爱去的山东馆子和北京小吃。现在却只有肯德基和麦当劳。

　　不管是音乐、建筑还是电影，我们的国家都在师夷长技，但却不能制夷。

　　回到窦唯的音乐，忽然觉得他跟bob dylan类似，不愿被打上标签。一直在追求自己喜欢的音乐，只是在大陆无法获得像鲍勃一样商业上的成功。

　　时代已经变了，我们还能说点什么？不外是保持清醒，有自己的判断吧？
