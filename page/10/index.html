人生·南北

发表于 2013-08-28 | 分类于 随笔

有一个同事辞职了，自己去开淘宝店了。
在来南方之前对南方的认识基本上属于空白，只是在新闻联播里看到每年过年的时候有巨多的人从广州往回走。
后来一个朋友说南方特别好，说将来一定要去南方发展，我当时顽固地觉得自己会在北方一直呆着，绝对不会去南方。
后来朋友倒是在北方安定，我却来到了南方，这大概就是现实或者造化。
南方是藏富于民，这一点不假。北方的人民多依赖政治，多渴望考公务员进入体制，借权力谋膏粱。
记得高中的时候就有同学提出过一个问题：权重要还是钱重要？
提出这个问题的是一个煤二代，现在他已经在日本读完了一个硕士，在英国读第二个硕士学位。
南方的人勤劳，但不仰仗于政治或者体制，大家都想的是要自己创业，制造属于自己的实体经济，而不是把自己的未来寄希望于权力的躯壳。
因为权力也会腐朽，钻营政治可能比搞好一个民营企业更加费心费神费力。
这也许就是南北的差距。
当然还有更多的人，像那个同事，没有屈从于政治，或者经济，只是想自由自在。
不自由，毋宁死。
政治，经济，自由，人生境界的不同，选择不同，坚持自己的选择。

生活琐事（三）

发表于 2013-08-25 | 分类于 随笔


　　商业写字楼群是人流密集的地方，一到中午各路白领都出来觅食，故而十字路口周边便成了黄金地带。小商贩是这里的常客，时令水果、小吃也能让人们换换口味。

　　这天很热，只有一位大叔坐在台阶上，身后大楼的影子遮住了他和他跟前的莲蓬。用竹篾编就的篓子装满了货品，倚靠在竹篓的白纸板上标着价钱——十元三个。

　　人行道上的行人匆匆走过，一把蒲扇在大叔手中摇晃地悠哉。

今天

发表于 2013-08-22 | 分类于 随笔

今天看到薄熙来穿着白衬衣受审，好像又有点可怜他，忘了他做了多少恶，有多少张扬跋扈权力金钱交易的往事。
这是原则性问题。
他受过苦，他爹翻身了，他就开始吃着他爹的政治资本一路高升，为什么偏偏是他落马？其中的究竟看多少期经济学人恐怕也知道不了。
这就是政治。和我们没什么关系。

热

发表于 2013-08-10 | 分类于 随笔

天天四十多度，不开空调根本不能活，早上听广播说企业限电保障空调用电，如果停电了，我们都得逃荒。
早上在车上看到晨练的人一如既往，西湖的水汽朦胧了远处的山，周六的早上不是很堵，钻进空调房成一统云端里看外面热气厮杀。
就这么一天又一天，忘了去年是怎么度过夏天，好像是海葵来过，也没有怎么样，生活就是这样，还能怎样？
在冬天期待夏天的温暖，期待来的却是烧烤一般的炙热。总是会问前路在何处，也许在那凉爽的家乡附近处。
不喜欢生活被别人干涉，自己爱怎么地怎么地，就像我管不了这热，只能管好自己，谁还明白这道理？
话说多了一点用处没有，生活就是劈头盖脸的忍耐，而且永远在脚下的路上。

生活琐事（二）

发表于 2013-08-06 | 分类于 随笔


　　正午，太阳毒得让人窒息。走至一个机票代售点，空调造出的冷空气从半敞开的金属玻璃门徐徐吹出。

　　一位打工者蹲靠在店面的窗台前，正用一次性筷子从塑料碗中夹起米线，额头浸出的汗水顺着脸颊慢慢滚落，在地砖上溅出一个圆，差一点就落在碗里。

　　我没有放慢走路的步伐，走出几米远后回望，发现在这位打工者身边的窗沿上搁着瓶可口可乐，标签上写着“高富帅”。

改变了的

发表于 2013-08-02 | 分类于 随笔

不知不觉，已经改变了许多，这些改变有大有小，有外在的有内心的，也许是每个人都会经历的过程。
1 不再向往大城市了。作为一个村里长大的娃从小就对城市有天然的向往，从对LF工贸文具柜台里粉色草莓橡皮擦的好感到在TY念书时对北京热烈的喜爱，到现在好像都熄灭了，只剩对长距离通勤的倦烦和挤八通线留下的不美好记忆。
2 不再向往做卓越的精英。活着活着路好像越来越窄了，少时的想法早已触不可及，人生是艰难的，在各扇大门关闭各种可能性灭绝之后，梦远了，只是低头日日做工而已。
如此种种，每天都会继续，最后会变成怎样的人，我也不懂自己了。

新东京物语——《东京家族》观后

发表于 2013-07-27 | 分类于 随笔


　　翻拍自1953年上映的黑白电影《东京物语》，60年后的今天导演山田洋次为我们带来《东京家族》。

　　不知为何，观看影片至十几分钟，想起动画片《我的邻居山田君》，也许是简单对话中不经意流露出的美好亲情。让人印象深刻的是，虽然早已知道日本人非常注重礼节却无概念，在影片中直观的感受到这一点。不论同辈抑或长晚辈之间都相敬如宾，但不会阻隔亲人之间的情感。

　　故事很简单，老两口结伴去东京这座繁华的大都市，在三个儿女的家中轮流小住，但完全没有“家”中舒服的感觉。当中，影片还不忘黑中国人，深夜在高级酒店的吵闹让无处可去只能在酒店休息的老两口深感无奈。目睹了在东京为生活奔波不易的儿女，老太太突然在大儿子家中晕倒住院抢救，又把散落在东京的儿女们聚在一起，还包括小儿子的女友。最后，老爷子回到家乡，把妻子留下的手表赠送小儿子的女友，接受了她，也对没有固定工作的小儿子放了心。

　　导演把60年前战后东京的相似故事，摆到了2012年的当下。城市化的进程，让养老成为社会问题，也让就业成为社会问题。反观中国，也面对同样的问题，自己也是。

　　浮躁的国内电影市场，鲜有这样温暖细致的影片，可能拍出来也不会有太大的市场，赔本的买卖估计没人会做。能回忆起来的相似影片只有《我们俩》，还记得是在CCAV的电影频道。

　　影片的平铺直叙，简单的故事以及对话，留给我的尽是温暖和思考，更印证了那句话，平平淡淡才是真。

　　回头补上《东京物语》，对比来看，希望会有更深切的感受。
