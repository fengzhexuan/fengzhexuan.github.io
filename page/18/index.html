城市里

发表于 2012-08-27 | 分类于 随笔

　　小时候，对城市没有任何概念——生长在小镇上的我不会想到外面的天空。

　　也许是电视给了我最初的启蒙，2007年，第一次去北京的时候，我就是那种感觉，走进了电视剧里的世界。90年代中央电视台播放的各种电视剧给了我城市最初的观感。那时晚上在王府井，小雨刚刚淋过，黑黝黝的夜色里，商场的灯透着稀疏的光，像极了某个电视剧里的场景，马路牙子边上的积水倒映着君悦大酒店的玻璃幕墙，喷泉好像在喷水又好像没有在喷水，一切影影绰绰漂浮不定在我的记忆里。

　　那时候还去了公主坟，对城乡结合部这个词有了初步的认识，只是当时我没有想到，原来那远远不是城乡的边界，甚至连后来熟悉的东五环，都不是。

　　城市好像是外来务工人员居无定所跌跌撞撞的一个大场所，他们在这里找不到自己的位置。周末的时候去外面逛，逛累了随性走进旁边的小区里，坐在椅子上那感觉真像个流浪汉。小区的居民是主人，我们，过客？还是什么都不是？

　　又得说回电视，这回是新闻联播，新闻联播里的北京好像永远都是蓝天白云风和日丽，钓鱼台紫光阁大会堂里的灯光瓦数总是恒定得温暖。可是当你真的踏上这片梦想中的热土，你才明白那些都与你无关，你甚至都不是政客游戏里的筹码。

　　漂泊总是和高房租高房价等字眼无奈地联系在一起，城市原来不是具体的一个地点，城市实为贯彻一种生活方式，挣钱养家吃喝拉撒，抛弃掉什么买房置业儿女绕膝的幻想，仅为苟存。

　　然而广袤的乡野也不会有我们的出路，虽然坐在公交车望着楼群放空的时候，会想到小时候早早起来去自家田里，回来的时候塑料凉鞋上沾满了透明的露水，脚丫上清凉的感觉，招摇着晨采的野花，手里淡淡的芬芳。

　　古人云：“衣沾不足惜，但使愿无违”。愿我们都记得出发的原因，继续笃定地漂泊。

信息架构与用户体验设计的暧昧关系

发表于 2012-08-26 | 分类于 随笔

　　最近一直在思考这样一个问题：信息架构与用户体验到底是什么样的关系？

　　这两个都是非常熟悉的概念，维基百科是这样介绍信息架构的：是在信息环境中，影响系统组织、导览、及分类标签的组合结构。信息架构会影响信息的可用性和可寻性（Findability），除此之外，仍有许多因素会影响，例如设计经验的累积、图形的设计、人机间的互动、知识内容的管理……

　　可以用一句话来概括他们的关系：信息架构是骨骼，用户体验则是肌肤与情感。

信息架构注重结构
　　信息架构这一概念远远早于互联网，而互联网则将之重要性提到了前所未有的高度。信息架构把复杂的信息经过有效的组织后更好的呈现给用户，通常可采用以用户为中心的设计方法来实现，譬如可用性测试、角色研究、用户流程图等（嗯……貌似这些也是些用户体验设计的方法）。优秀的信息架构将使产品拥有健壮的骨骼。
用户体验设计注重情感
　　用户体验设计比信息架构更进一步，它让产品更易用。为了对目标受众产生有凝聚力的、可预见性的、可取的影响，逐渐形成了以用户为中心的设计方法。通过分析用户的上下文情境以及自然行为的背景和故事，在用户情感的层面上做出设计。让产品在健壮的骨骼上上生长肌肉，甚而赋予产品情感。

　　优秀的信息架构让产品简单易用，而优秀的用户体验设计让产品具有生命力并对用户留下持续的深刻印象。比如，SketchBox与Paper，相同的功能给人的感觉却是不一样的。面对SketchBox你可能会说，它挺好用的，而面对Paper你可能会说，这个产品够酷。

　　国外会有专门的信息架构师来进行信息架构的工作，国内很少有公司专门设立这样的职位，更多的由交互设计师进行。交互设计经常会陷于对某些细节的纠结与反复，而信息架构也是一个交互设计师分内的事情。

　　一个好的产品必将有骨骼、有血肉、有情感。

我的奥运记忆

响应式设计的7种工具

发表于 2012-08-13 | 分类于 随笔

　　响应式设计在今年真是风生水起，国外的朋友已经在这方面做了很多尝试与研究。本文向大家介绍几种浏览器兼容性不错的工具，让读者在响应式设计的路上越走越深。

Adaptive Images
　　目前越来越多的用户通过移动设备来浏览网站，而他们通常处于2G或3G网络。除了为小屏幕设计与开发的一些挑战外，如何重新调整图片大小以适应小屏幕也同样必须纳入考虑范围。Adaptive Images就是这样一种服务端的PHP工具，它根据检测到的屏幕大小来帮助你调整图片。
FitText
　　刚说过了图片，现在该文字了。你是否为让标题适应父元素的宽度而头疼不已？这下你走运了，这个jQuery插件能够让你的文字自动适应屏幕的宽度。
PhotoSwipe

　　这个工具能让你的Gallery不论在桌面还是移动设备上看起来都能完美无瑕，它能让你的网页看起来跟原生的应用一样。忘了告诉你，还支持触屏的手势操作哦~
Less框架
　　这个就是大名鼎鼎的Less框架了。当你的网站很复杂时，通过使用同一个网格（Grid），只需调整宽度、字体大小等一些小细节便能让网页的元素得到重用。
Skeleton
　　啊哦，又一个框架，细节就不多说了，它基于一个轻量级的960网格系统开发。
响应式设计测试工具
　　测试响应式设计是个头疼的事情，虽然可以通过调整浏览器的宽度来查看，但无法一目了然，有了这个工具就碉堡了。不过这个工具还是很简单的，相信读者自己写个本地的出来也木有问题。
Inuit CSS Framework
　　同样是个CSS框架，它使用起来非常的灵活，能让你方便的构建出自适应的网站。

　　如果你还有其他更多的关于响应式设计的工具，欢迎交流～

吃货成长史

发表于 2012-08-10 | 分类于 随笔

（图片来自Dribble）

　　谨以此文追溯一下一个吃货的进化史。

　　小时候，我只从别人那里知道自己是个挑食的人，饭的成色不好，或者味道差，我便没有了胃口。所幸身为厨师的爷爷技艺一流，妈妈和姥姥的厨艺也很棒，于是我度过了一个饱食三餐的童年并养成了一个挑剔的胃。

　　上初中也基本上是在别人家里吃，开始通勤的时候，晚饭没有着落就在学校外面吃几串炸蘑菇买个毛毛虫面包或者干脆面或者乡巴佬卤蛋解决，对了，还有学校大门旁边的“西安凉皮”，里面的担担面和鸡肉米线是大家在下午放学以后不回家吃饭的不二选择。还有下了晚自习以后，学校门外的豆腐串，切了薄片的冻豆腐炸了一下又在调料锅（还是铝盆？）里煮了煮，便是经过繁重脑力劳动以后的珍馐。

　　上了高中以后，开始了受虐的食堂生涯，不夸张的说，有一段时间我走进那大礼堂般的食堂后便开始干呕。那些在大铝盆里用铁铲搅拌的菜，我早已忘了有哪些花样，只记得面食窗口永远火爆——西红柿鸡蛋面里细细的豆芽有时竟像头发，还吃出过各种奇怪的物件，比如自行车的内胎。另外，高一的时候对两块五一只的鸡腿一块五一串的鸡排颇有好感，后来吃腻了也味同嚼蜡。后来的后来，我就不再吃食堂了。转而奔向校门外的“西安凉皮”们，正午时分，家家坐满了等着大快朵颐的学生，炒一碗担担面两块五，鸡肉米线是三块，那一家挨着一家好像永远都吃不完。学校后门的斜对面有个精华饭庄，去那里吃过一两次，炒土豆丝什么的，对于饱受食堂蹂躏的胃来说已是无上的犒赏。那时候的饮食生活相当的贫乏，除了吃学校周围的担担面和米线便是水厂对面的马兰拉面，跟爸一起去吃的时候还总会点尖椒皮蛋，再呼噜呼噜吃完一碗面，甭提多爽快。还有鼓楼东面（还是解放路上？）的李点拉面，这家拉面后来搬到了中学的西面，吃过几次以后就没再去了。那个时候、那个地方的餐饮还没有什么想象力，觉得最高级的居然是银河超市市委店对面的一家店，把一碗大米倒扣在盘子里，旁边点缀一切为二的卤蛋和生菜，好像还要标价八块。啧啧，真贵。等到高二还是高三的时候，学校的后门开始有了“夜市”，那时候莲花巷还没有拓宽，窄窄的马路上各种小三轮，有卖臭豆腐的有卖炸鸡柳的其他的不好意思想不起来了，常吃的就是臭豆腐和炸鸡柳，那豆腐不过是寻常的炸豆腐，没有什么臭味，关键在装在暖瓶里的调味汤，再混上香菜有独特的味道，也许每次都是冲着那碗汤去的，本末倒置罢。卖炸鸡柳的总是抓一把鸡柳放在盘子里称一称，那时好像总是要两块钱的，等到鸡柳入锅，我站在旁边发呆放空一会儿再左张右看一会儿，鸡柳就炸好了，我也就大吃一通，抹一抹嘴心满意足地走回小屋去。后来的夜市开始提早，供应晚饭，那时常吃的是鸡蛋灌饼夹火腿，谈不上有多好吃，但在那时却是独一无二的滋味，那面酱的味道，现在还记得。

　　上了大学以后，开始经常吃食堂，从军训伊始，我就开始经常去食堂吃饭，吃完饭还爱赖在食堂里听听那喧闹。从那时起，我认得了蟹棒这种东西，开始在大嚼特嚼热汤面以后喝一瓶冰镇的可乐，开始吃盖浇饭——最爱的品种是令德二楼的牛肉豆腐，开始学会了催饭，开始学会了下馆子，开始吃自助火锅自助烤肉，开始办肯德基的学生优惠卡，在开始新一轮集体生活之后也学会了吃独食——在上完了双学位课以后，特别疲惫的我心心念念的只有一件事，那就是去吃一个汉堡包。大学里面的各种小店也逃不过那时饥饿的我们，一开始是六子米皮，后来是土豆粉，再后来是河东面食，再后来是家乡面食，快毕业了是香菇滑鸡煲，还有总是AA的苏大姐火锅，还有许西村头的顶禾炸鸡，以及紧邻宿舍区的第一家豆浆，那一年冬天的早餐全部都是它，以及超市旁边的兰州牛肉面，老板娘不知道为什么没有小手指头。

　　到了帝都以后，吃的事业更加蒸蒸日上，东奔西突走南闯北，只为那微博上的美食。那时学会了按图索骥，却还是在吃完螺蛳粉以后，坐反了回去的车。为了等卤煮店开门，在北新桥附近的胡同里游荡了一下午。在北科北门吃串总是坐靠窗靠墙的那个位置，在汉丽轩的十里河望着团购汹涌的食客怒火滔天，萨莉亚的意面不同的店都千差万别，还有麦当劳的岩烧鸡腿堡，吃腻的呷哺呷哺，等等等等。

　　以上种种吃，不过是表象，也许是年轻，还有发达的味蕾，也许是觉得其他的东西太虚，只有拾掇到胃里，才能温暖自己，进而充实心灵。

炸酱面

发表于 2012-07-29 | 分类于 随笔

材料：黄豆酱 150g；三肥七瘦猪肉 150g；黄瓜一根；面条 750g。

1.首先将猪肉洗净，切至肉末状，黄瓜洗净，切细丝。

2.用凉水化开黄豆酱。

3.热锅倒入油，待油熟后，倒入肉末翻炒并倒入少许醋，待七成熟时将黄豆酱倒入，迅速翻炒。最后加少许酱油，因为黄豆酱本身既咸又上色。待翻炒均匀后关火，至此酱即成。

4.煮面乘盘。

5.在面上摆好黄瓜丝，浇上炸酱。

6.搅拌后好好享用吧~
